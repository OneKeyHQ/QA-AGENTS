## 一、真实固件问题（真机中文环境同样出现）

### 1.【高·系统性】中文换行没有"避头尾"规则

标点（？ 。 "）被折到行首、单字孤行。根因：LVGL 断行对 CJK 标点无禁则处理，一处修复全局受益。

| 页面 | 现象 |
|------|------|
| settings_turbo_confirm | 标题"启用 Turbo 模式 / **？**"；正文"…加快签名 / **。**" |
| settings_safety_disable_2 | 标题断成"永久禁用安全检 / **查？**" |
| settings_sk_get_counter | 标题"…计数 / **器**"孤字；正文"…吗 / **？**" |
| settings_ms_warning / settings_reset_erase | 勾选项句号"。"孤行 |
| fingerprint_adding / onboarding_keep_it_safe | 句号悬挂行首 |
| poweroff_findmy_serial | 闭引号 **"** 折到行首 |
| onboarding_backup_mode_pick / onboarding_device_verified | "式。"/"绪"单字孤行 |

### 2.【高·系统性】签名/交易流程大面积英文——翻译已存在但代码未接 i18n

- **C 层**：`ui/templates/txdetails_keys.h::txdetails_key_to_label()` 硬编码返回英文 `"To"/"From"/"Network"/"Maximum Fee"...`。语言包中对应中文（至/网络/最高手续费，tag 961/942/938）已存在但未被使用。
- **MP 层**：frozen 引擎（`app/<chain>/**/layout.py`）标题与提示硬编码英文：`sign("Send", ...)`、`"Warning"`、`"Unknown Transaction"`、`"This token is not recognized by the hardware wallet. Please verify carefully."`、钓鱼风险提示 `"Risk of phishing & blind signing..."` 等。
- 现状：签名页仅模板控件（概览/详情/继续/滑动签署）为中文，标题、字段标签、**安全警告**全英文。安全警告不翻译对中文用户风险最高。
- 证据：`eth-erc20.png`、`sol-spl-unknown.png`、`eth-approve.png` 等。

## 新增真实固件问题

### 🔴 1.【高】KeyTag 打点图正/背面图片资产用反（备份错误风险）

`settings_keytag_backup_chart`（12 词）：页面标注"正面（#1 - 12）"，但打点图行号显示 **13–24**。根因：`ui/assets/asset_paths.h:863-864` 的 legacy 别名疑似写反——`ASSET_GRAPHIC_KEYTAG_FRONT → KEYTAG_24`、`ASSET_GRAPHIC_KEYTAG_BACK → KEYTAG_12`。页面逻辑正确（page 0 用 FRONT），图却是背面行号。**用户照图打点会打错行**。证据：`settings_keytag_backup_chart.png`。

### 🔴 2.【高】错误提示硬编码英文（未接 i18n 的新实例，中文翻译已存在）

- `fingerprint_enroll_result.c`：`"Fingerprint Timeout"`、`"Failed to Add Fingerprint"`、`"Try Again"` 全部 C 层硬编码英文，而语言包 tag 161/163（添加指纹失败/指纹超时）闲置。指纹失败/超时是用户必见的错误页。
- `settings_fingerprint.c` / `fingerprint_adding.c`：`"Serial Number Not Set / Please contact the developers."` 弹窗硬编码英文。
- 证据：`fingerprint_failed.png`、`fingerprint_timeout.png`、`fingerprint_adding--click06.png`。

### 🔴 3.【高】英文源文案残缺："Connect the device: ."

连接钱包教程第 2 步显示"**连接设备: .**"——tag 714 的 en 源文案本身就是 `'Connect the device: .'`（变量丢失只剩 ": ."），全语言跟着残缺；正确变体 tag 1065 `'Connect the device: %s'` 未被使用。证据：`tips_app_tutorial--after-clicks.png`。
