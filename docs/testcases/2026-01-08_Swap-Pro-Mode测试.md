# Swap Pro Mode 模块测试
> 生成时间：2026-01-08

## 测试场景列表

**说明**：
- **平台限制**：仅支持移动端（iOS/Android），桌面端不支持
- **交易方向**：购买（Buy）/ 出售（Sell）
- **交易模式**：市价单（Market）/ 限价单（Limit）
- **支持网络**：BSC、Solana、Base、Ethereum、Polygon、Arbitrum、Avalanche、Optimism、SUI、X layer、Sonic、Linea、Cronos Mainnet、Plasma、Ton、Tron 等（详见文档）
- **网络验证要求**：以下测试用例需要在所有支持的网络上验证相同的测试场景
- **特殊说明**：
  - 部分网络不支持 topholder（如 Sonic、Linea、Cronos Mainnet、Plasma、Ton）
  - 部分网络不支持持仓（如 Sonic、Plasma、Ton）
  - 限价单相关测试详见 `docs/testcases/2026-01-08_Limit-CowSwap渠道测试.md`

## 自动化实施方案（P0/P1）

### 1. 单元测试（Unit）重点
- **入口跳转逻辑**：验证 Token Details 页面入口跳转到 Swap Pro Mode 的逻辑
- **FromToken 类型验证逻辑**：验证主币和代币的不同处理逻辑（主币无需授权，代币需要授权）
- **交易模式切换逻辑**：验证市价单/限价单切换时的界面变化和数据保留逻辑
- **代币信息切换逻辑**：验证代币选择弹窗和代币切换逻辑
- **市场数据更新逻辑**：验证 WebSocket 数据更新和界面刷新逻辑
- **询价计算逻辑**：验证询价接口调用和数据显示逻辑
- **余额计算逻辑**：验证余额显示和余额不足判断逻辑
- **持仓列表排序逻辑**：验证持仓列表按价值排序的逻辑
- **网络切换逻辑**：验证网络切换时代币列表、余额、市场数据的更新逻辑

### 2. 端到端（E2E/API）重点
- **入口测试**：Token Details 页面入口显示和跳转
- **交易方向测试**：购买/出售切换和界面更新
- **交易模式测试**：市价单/限价单切换和功能验证
- **代币信息测试**：代币选择弹窗和代币切换
- **市场数据测试**：Market cap、24h Vol、Liquidity、Holders 显示和更新
- **价格信息测试**：当前价格、价格变化、Price / Value 列表显示和更新
- **买入卖出比例测试**：B/S 比例和交易量显示和更新
- **交易输入测试**：数量输入、快速选择、限价单价格输入
- **询价信息测试**：Est. Receive、Trading Fee、Network Fee 显示和更新
- **余额和钱包测试**：余额显示、钱包地址显示、钱包切换
- **滑点设置测试**：自动滑点和手动滑点设置
- **我的持仓测试**：持仓列表显示、当前代币筛选、持仓点击切换代币
- **未成交订单测试**：Open order 列表显示、取消限价单
- **网络切换测试**：网络切换和界面更新
- **不支持提示测试**：网络不支持、代币不支持、BTC 不支持交易提示
- **交易执行测试**：市价单和限价单交易执行流程

## 测试数据准备

- 账户类型：软件钱包、硬件钱包、私钥账户、外部账户

- 主币：ETH、BNB、MATIC 等（用于测试主币作为 FromToken）
- 稳定币：USDC、USDT、DAI 等（用于测试代币作为 FromToken）
- 其他代币：选择流动性较好的代币（用于测试代币作为 FromToken）
- **特殊说明**：BTC 不支持交易，需要验证不支持提示

### 测试网络要求
- 支持的网络：X layer、Sonic、Linea、Cronos Mainnet、Plasma、Ton、Tron
- 需要在每个网络上验证相同的测试场景

## 注意事项

1. **平台限制**：Swap Pro Mode 仅支持移动端，桌面端不支持，测试时需要使用移动端设备或移动端模拟器
2. **网络特性**：部分网络不支持 topholder 或持仓，测试时需要注意这些限制
3. **限价单测试**：限价单的详细测试用例请参考 `docs/testcases/2026-01-08_Limit-CowSwap渠道测试.md`
4. **市价单测试**：市价单的详细测试用例请参考 Swap 模块的测试用例
5. **数据实时性**：市场数据、价格信息等通过 WebSocket 实时更新，测试时需要验证数据更新的实时性和准确性
6. **多链支持**：Swap Pro Mode 支持多链，测试时需要验证网络切换和跨链数据的一致性
