## 发现的问题

### P0 问题
1. **收藏功能需要登录**：收藏 Token 功能需要用户登录或启用云同步，未登录状态下无法完成收藏操作
   - **错误信息**：`OneKeyError: Cloud sync is not enabled`
   - **影响**：无法完整测试收藏功能的核心流程
   - **建议**：提供未登录状态下的收藏功能，或明确提示用户需要登录

### P1 问题
1. **搜索功能位置不明确**：页面顶部有搜索图标，但未找到搜索输入框
   - **影响**：无法测试搜索列表的收藏功能
   - **建议**：确认搜索功能的触发方式或位置

2. **性能脚本未注入**：未在页面加载前注入性能采集脚本
   - **影响**：无法获取 Core Web Vitals 等性能指标
   - **建议**：在测试开始前注入性能采集脚本
