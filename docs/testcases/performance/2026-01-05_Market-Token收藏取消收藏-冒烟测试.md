## 发现的问题

### 1. 云同步功能未启用

**问题描述**：执行收藏/取消收藏操作时，控制台出现错误信息：
```
OneKeyError: Cloud sync is not enabled
```

**影响范围**：可能影响跨端数据同步功能（场景5(d)）

**建议**：
- 需要用户登录并启用云同步功能才能测试跨端同步场景
- 或者使用已登录的测试账号进行完整测试

### 2. 部分场景需要特殊环境

**问题描述**：以下场景需要特殊环境或人工验证：
- 场景6：网络异常与容错测试（需要模拟网络异常）
- 场景7：多端适配测试（需要 Android/iOS/Desktop/Extension 环境）
- 场景8：边界与异常场景测试（需要测试收藏数量上限等）

**建议**：
- 网络异常测试：使用浏览器 DevTools 的网络节流功能
- 多端适配测试：使用真实设备或模拟器
- 边界场景测试：需要准备测试数据（如 100+ 收藏 Token）

### 3. 性能指标采集不完整

**问题描述**：
- LCP 和 FID 未采集到（可能因为脚本注入时机或页面已加载完成）
- 运行时性能指标（内存使用）未采集

**建议**：
- 在页面加载前注入性能采集脚本
- 或刷新页面重新采集完整性能数据
