# Wallet - Stellar (XLM) 软件钱包测试

## 测试概览

**测试目标**：验证 Stellar 链在软件钱包下的基础转账、Token 管理、dApp 连接、私钥导出等功能。

**测试范围**：
- XLM 原生转账（新账户创建 vs 老账户转账）
- Stellar Asset Token 转账与激活
- Contract Token 转账
- Token 添加与搜索逻辑
- 账户激活与保证金机制
- dApp 连接与 Swap
- 私钥导出与验证

**测试环境**：Mainnet
