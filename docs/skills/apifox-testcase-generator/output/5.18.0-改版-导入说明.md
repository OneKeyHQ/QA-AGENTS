## 🔧 参数传递规则

### ✅ 直接传参（硬编码）

固定参数直接写在请求中，无需配置：

| 参数 | 值 | 说明 |
|------|-----|------|
| `baseUrl` | `https://api.onekey.so` | API 地址 |
| `networkId` | `sol--101` | Solana 主网 |
| `X-Onekey-Request-Version` | `5.18.0` | 客户端版本 |
| 请求头 | `5.18.0` 等 | 版本等信息 |

### 🔄 动态传递（环境变量）

接口间需要传递的动态数据，通过 Test 脚本自动写入环境变量：

| 变量 | 提取接口 | 使用接口 | 提取代码 |
|------|---------|---------|---------|
| `marketAddress` | 01-获取 Market 列表 | 后续所有接口 | `pm.environment.set('marketAddress', ...)` |
| `provider` | 01-获取 Market 列表 | 后续所有接口 | `pm.environment.set('provider', ...)` |
| `networkId` | 01-获取 Market 列表 | 后续所有接口 | `pm.environment.set('networkId', ...)` |

## 📊 断言说明

每个用例包含以下断言：

### 基础断言
```javascript
// HTTP 状态码
pm.test('响应状态码为 200', function() {
    pm.response.to.have.status(200);
});

// 响应时间
pm.test('响应时间小于 3000ms', function() {
    pm.expect(pm.response.responseTime).to.be.below(3000);
});

// 业务状态码
pm.test('业务状态码为 0', function() {
    pm.expect(jsonData.code).to.eql(0);
});
```

### 数据结构断言
```javascript
// 数组验证
pm.test('markets 是数组且不为空', function() {
    pm.expect(jsonData.data.markets).to.be.an('array');
    pm.expect(jsonData.data.markets.length).to.be.above(0);
});

// 字段验证
pm.test('包含必要字段', function() {
    pm.expect(jsonData.data).to.have.property('xxx');
});
```

### 环境变量提取
```javascript
// ★ 提取动态数据到环境变量
if (jsonData.data.markets && jsonData.data.markets.length > 0) {
    pm.environment.set('marketAddress', jsonData.data.markets[0].address);
    console.log('★ 已写入环境变量 marketAddress');
}
```
