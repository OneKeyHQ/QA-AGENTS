## 前置条件与测试数据

1. 使用 Desktop TF 包：`/Applications/OneKey-3.localized/OneKey.app`。
2. 测试前通过 TestFlight 检查 OneKey 最新版本：
   - 仅允许更新到 `6.x` 开头版本。
   - 如果 TestFlight 最新版本为 `900.x` 开头，跳过更新并记录跳过原因。
   - 该限制只适用于本 nightly checklist，不影响其他 checklist。
3. 更新后启动 OneKey 并确认 CDP 可用：`http://127.0.0.1:9222/json/version`。
4. UI 异常判定：
   - 页面白屏、主要区域空白。
   - 列表无数据但未显示明确空态。
   - loading 长时间不消失。
   - 弹窗/遮罩残留阻塞主流程。
   - 页面切换后数据显示错位、明显断层或异常报错。
5. 本用例允许使用录制中的公开测试数据：
   - App 密码：`1234567890-=`
   - 邀请码：`VIP999`
   - 助记词：`abandon` x 11 + `about`
   - 观察地址 / 域名：`ONEKEY.eth`
   - 私钥导入：使用本流程中从公开测试账户导出的私钥，不使用有资产账户。

---

## 清理规则

- 合并同一页面连续重复点击，只保留一次有效导航或一次 UI 展示检查。
- 保留用于检查列表数据、滚动加载、页面非空展示的滚动步骤。
- 删除无状态变化的重复弹窗背景点击、重复 market 空白区域点击。
- 对敏感页保留“进入、验证、确认、返回”关键节点，不保留无意义重复点击。
