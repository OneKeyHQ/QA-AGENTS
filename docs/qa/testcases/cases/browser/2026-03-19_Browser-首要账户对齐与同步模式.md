# Browser - 首要账户对齐与同步模式

> 生成时间：2026-03-19
> 更新时间：2026-03-19（账户对齐/独立模式/始终使用钱包账户 + 异常鲁棒性）

## 测试范围说明

**测试覆盖要求**：
- 对齐首要账户：默认「将 dApp 账户对齐至钱包」、「独立模式」、「始终使用钱包账户」
- 多 DApp 标签页连接的账户/网络隔离逻辑
- 账户派生路径：Solana 与 EVM 路径 BIP44 <-> Ledger Live
- 断开重连与切换中断等异常鲁棒性
