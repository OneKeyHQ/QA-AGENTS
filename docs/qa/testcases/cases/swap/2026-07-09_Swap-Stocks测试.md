> 生成时间：2026-07-09
> 规则文档：`docs/qa/rules/swap-rules.md`、`docs/qa/rules/market-rules.md`、`docs/qa/qa-rules.md`
> 需求文档：`docs/qa/requirements/Swap-Stocks.md`
> 测试端：Android / iOS / Desktop / Extension / Web
> 关联资源：用户提供的移动端 Stocks 交易页、股票代币选择页、桌面端 Stocks 页面截图

## 测试范围说明

**功能范围**：Trade 下股票（Stocks）分类入口、移动端 Tab 布局、股票交易面板、Buy / Sell 默认逻辑、设置差异、My position、股票代币切换与搜索、Review Order、交易历史、后端错误控制、Desktop K 线与 Market Data。

**覆盖重点**：
- Stocks 分类先独立接入 Trade，不等待 `Swap & Bridge` 合并
- 当前范围是 BNB Chain 维护股票列表 + 固定渠道 `1inch Fusion` + 固定稳定币 `USDC / USDT`
- Stocks 模式统一走 REQ，设置页不显示滑点
- K 线入口跳转 Market Token details，Desktop 额外展示 TradingView 与 Market Data

---

## 前置条件

1. 已登录 HD 钱包 / HW 钱包，Trade 页面可打开，BSC 网络可用
2. Dashboard 已配置 Stocks 可买卖稳定币列表，默认顺序至少包含 `USDC / USDT`
3. 已准备三组余额数据：`USDC > USDT`、`USDT > USDC`、`USDC = USDT = 0`
4. 可通过测试环境 / Mock 控制股票列表、报价、交易控制、休市状态、Perps 同标的、倒计时、Market 数据
5. 已准备一组仅持有股票代币的账户、一组仅持有普通代币的账户、一组无相关持仓的账户

|---|---|---|---|
