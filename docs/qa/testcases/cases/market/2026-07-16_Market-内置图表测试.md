# Market - 内置图表测试

> 生成时间：2026-07-16
> 规则文档：`docs/qa/rules/market-rules.md`、`docs/qa/qa-rules.md`
> 需求文档：`docs/qa/requirements/Market-内置图表.md`、`docs/qa/requirements/Market-图表.md`
> 测试端：Android / iOS / Extension / Desktop / Web
> 关联资源：现网 TradingView 图表作为对照参考；既有通用图表用例 `docs/qa/testcases/cases/market/2026-03-26_Market-图表.md`

## 测试范围说明

**功能范围**：Market Token 详情页由内置图表替换现有 TradingView 后的首屏加载、周期切换、OHLC 对照、技术指标、十字光标、缩放平移、实时刷新、空态错态与跨端渲染一致性。

**覆盖重点**：
- 内置图表替换后，详情页图表区域可用，不出现白屏或崩溃
- 与 TradingView 对照时，只比较同一标的、同一周期、同一时区下的已收盘 K 线
- 指标、十字光标、缩放平移、长历史与断网恢复延续现有图表能力

---

## 前置条件

1. 已准备至少 2 个可稳定出图的主流 Token，覆盖高成交量与中等成交量场景
2. 已准备 TradingView 对照页，并可手动切到与 App 相同的标的、周期、时区
3. 默认联网；异常场景可手动断网 / Mock 图表接口失败
4. 已准备助记词钱包 / 观察账户；其余账户类型按端能力抽测
5. 若需验证指标口径，已确认 EMA / MACD / RSI / 布林带参数与 TradingView 对照参数一致
