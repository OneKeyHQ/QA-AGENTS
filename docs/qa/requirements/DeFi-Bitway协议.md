# DeFi - Bitway 协议（简单赚币新渠道）

> **App 版本**：<br>
> **测试端**：默认全端（本需求截图基于桌面端）<br>
> **需求来源**：DeFi 新增 Bitway 渠道（2026-07-20 用户提供 + 桌面端截图）

## 1. 渠道概述

- **协议**：Bitway — 兼容 Bitcoin 的权益证明 Layer 1 区块链，将链上 BTC 流动性与全球 DeFi 服务连接（标签：bitcoin / layer1 / staking / btcfi）
- **产品名称**：Bitway Absolute Return（按币种区分：Bitway Absolute Return(U) / Bitway Absolute Return(USDT)）
- **网络**：BSC（BNB Chain）
- **接入代币**：U、USDT（各自独立的认购/赎回入口）
- **凭证代币**：bwU / bwUSDT（详情页展示，可跳转区块浏览器）
- **协议信息**：成立时间 2023，团队成员 Shane Qiu 等，投资方 TRON DAO 等，审计 BlockSec 等，外链：网站 / X / Discord
- **钱包支持**：HD 与 HW 钱包均支持认购/赎回签名（HW 走设备确认）

## 2. 认购规则

| 项 | 规则 |
|---|------|
| 最小认购金额 | 10（10 U / 10 USDT）；低于最小值显示红色警告「最小金额为 10 USDT。」且「授权」按钮置灰 |
| 最大认购金额 | 无上限（仅受钱包余额限制） |
| 认购金额精度 | 18 位小数（USDT 与 U 相同） |
| 认购流程 | 两步：1. 授权 → 2. 认购（底部 Stepper 显示「1. 授权 → 2. 认购」） |
| APR | 单一 APR（如 5.00%），无多段拆分；详情页显示 高 / 低 历史区间与走势图 |
| 预估年收益 | `预估年收益 = APR × 认购金额`（同币种计价 + USD 估值） |
| 余额不足 | 显示「U/USDT 不足？」引导，提供「交易」「购买」按钮 |

## 3. 赎回规则（两种提现方式）

赎回 tab 输入金额后点击「赎回」，弹出「提现选项」弹窗，包含两个选项：

| 提现方式 | 到账时间 | 手续费 | 接收金额 | 限制 |
|---------|---------|--------|---------|------|
| 排队提现（默认选中） | 约 7 天后可领取 | 无提现手续费 | = 赎回金额 | 无 |
| 立即提现 | 立即到账 | 项目方收取 0.5% 手续费 | = 赎回金额 × 99.5% | 受闪电兑换池（闪付池）流动性限制 |

- **赎回金额范围**：最小 **0.00000001**（1e-8，USDT 与 U 两币种相同）；最大为当前持仓（可 100% 全部赎回）
- **立即提现流动性不足**：当赎回金额 > 闪电兑换池余额时，「立即提现」选项置灰不可选，并显示黄色提示：「闪付池余额不足。当前可用流动性为 X（币种），请等待资金到位，或使用排队提现功能。」
- **立即提现金额换算示例**：赎回 11 USDT → 立即提现接收 10.945 USDT（11 × 99.5%）
- 详情页「简介」区展示 **流动性** 与 **闪电兑换池余额** 两个独立数值（如 流动性 84.19K USDT / 闪电兑换池余额 2.96K USDT）
- **排队赎回中状态**：提交排队提现后，持仓行显示「赎回中」数量 + info icon（不计入已激活）；点击 icon 弹出明细，每笔排队赎回独立一行（金额 + 剩余天数，如「1 USDT 剩余 2 天」），底部文案「在上述时间之后，您认购的资产将可供领取。」；多笔独立排队各自 7 天倒计时，赎回中总金额 = 各笔明细之和
- **到期领取**：倒计时结束后需手动领取赎回金额（全额，无手续费）；领取时该笔对应的收益随赎回一并结算到账（奖励不可单独领取）

## 4. 持仓与收益

- 持仓入口：DeFi 首页「持仓」tab（赚币 / 借币 切换下的赚币侧），按渠道分组显示 Bitway 组及总金额
- 持仓行字段：已认购数量 +（USD 估值）+ 产品名、预计 24 小时收益、累计收益、已激活数量、赎回中数量（如有）、收益数量
- **认购成功后立即激活**：已激活数量随认购立即增加，无激活等待期
- **奖励结算规则**：奖励不可单独领取，将与资产赎回时一并结算（收益 info icon 的 tooltip 文案）
- **累计收益同步频率**：「累计收益」字段约 **1 小时同步一次**（非实时刷新）；可与 Bitway 第三方协议侧（官方站点）同持仓的收益数据对比确认准确性
- 持仓行提供「管理」按钮 → 打开认购/赎回管理弹窗（含 历史记录 入口）
