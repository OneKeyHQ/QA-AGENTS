# Perps - TradingView 图表功能与设置

> **模块**：Perps（合约交易）
> **功能名称**：TradingView 图表功能与设置
> **版本**：
> **测试端**：iOS / Android / Extension / Desktop / Web

---

## 📋 需求背景

Perps 合约交易模块集成 TradingView 图表，用户可以在图表上进行技术分析、查看持仓信息、自定义图表设置。本需求覆盖 TV 图表的核心功能与设置项。

---

## 📝 功能描述

### 1. 默认指标

- **默认指标**：成交量（Volume）
- 首次进入图表时，自动加载成交量指标
- 成交量指标显示在主图下方子图区域

### 2. 指标管理

#### 2.1 添加指标
- 支持添加多种技术指标（如 MA、EMA、MACD、RSI、BOLL 等）
- 添加的指标**本地持久化保存**，刷新页面、关闭浏览器后仍保留
- 指标配置（参数设置）同步保存

#### 2.2 指标收藏
- 支持收藏常用指标
- 收藏的指标显示在指标列表顶部或独立收藏分组
- 收藏状态本地持久化保存

#### 2.3 指标删除
- 支持删除已添加的指标
- 删除后，本地存储同步更新

### 3. 画图工具

- 支持多种画图工具（趋势线、水平线、斐波那契、矩形等）
- 绘制的图形**本地持久化保存**
- 刷新页面、关闭浏览器后，画图内容仍保留
- 支持编辑已绘制的图形
- 支持删除已绘制的图形

### 4. K 线时间周期

#### 4.1 预设时间周期
- 支持常用时间周期：1分钟、3分钟、5分钟、15分钟、30分钟
- 支持小时级别：1小时、2小时、4小时、8小时、12小时
- 支持日级别及以上：1天

#### 4.2 自定义时间周期
- 支持自定义 K 线时间周期
- 自定义周期本地保存
- 支持收藏常用时间周期（显示星形标记）

### 5. 图表叠加显示

#### 5.1 买卖点显示
- **在图表上显示买卖点**：开仓/平仓点位标记
- 可通过设置项控制显示/隐藏

#### 5.2 持仓信息线
- **当前持仓线**：显示当前持仓的开仓价格线
- 可通过设置项控制显示/隐藏

#### 5.3 委托挂单线
- **委托挂单线**：显示当前未成交的限价单价格线
- 可通过设置项控制显示/隐藏

#### 5.4 爆仓线
- **爆仓线**：显示当前持仓的强制平仓价格线
- 可通过设置项控制显示/隐藏

#### 5.5 设置项
- 提供统一设置面板控制以上内容的显示/隐藏
- 设置项包含：
  - 「在图表上显示买卖点」开关
  - 「在图表显示仓位和订单」开关（控制持仓线、委托挂单线、爆仓线）
- 设置状态本地持久化保存

### 6. 视图布局

#### 6.1 自定义布局
- 支持自定义图表视图布局
- 布局配置本地持久化保存

#### 6.2 恢复默认布局
- 提供「返回默认布局」设置项
- 点击后恢复图表到默认布局状态
- 清除自定义布局配置

## 📋 业务规则

1. **本地存储**：所有自定义配置（指标、画图、时间周期、设置项、布局）均使用 localStorage 或 IndexedDB 本地存储
2. **数据隔离**：不同交易对（如 BTCUSDC、ETHUSDC）的画图数据相互隔离
3. **性能要求**：指标计算、画图渲染不影响图表流畅性（FPS ≥ 30）
4. **跨端同步**：本地存储仅限当前设备，不跨设备同步（除非后续支持云同步）
