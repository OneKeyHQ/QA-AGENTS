# Market - 我的持仓需求文档

> **模块**：Market（行情）
> **功能名称**：我的持仓（My position）
> **版本**：
> **测试端**：Android / iOS / desktop / ext / web

---

## 1. 需求背景

在 Market 代币详情页的 History 区域，需要支持展示“我的持仓”与“Holders”两个视图。用户在不同账户类型下查看同一代币时，应看到一致且可对账的持仓数据与买卖点信息。

---

## 2. 功能描述

### 2.1 视图结构

| 规则项 | 规则描述 |
|-------|---------|
| 区域名称 | History |
| 顶部切换 | My position / Holders（数量） |
| My position 字段 | Token、Balance、Unrealized PnL、Total PnL |

### 2.2 有持仓场景

| 规则项 | 规则描述 |
|-------|---------|
| 持仓数量与估值 | 显示当前代币持仓数量与估值 |
| 未实现收益 | 显示 Unrealized PnL 数值与百分比 |
| 总收益 | 显示 Total PnL 数值与百分比 |
| 买卖点 | 在 K 线图显示买卖点，鼠标悬浮可展示详情 |
| 数据一致性 | 与 OKX Dex 对比数据一致 |

### 2.3 无持仓场景

| 规则项 | 规则描述 |
|-------|---------|
| My position 字段 | 不展示持仓字段内容（显示占位或不显示） |
| 买卖点 | 不显示买卖点 |

---

## 3. 前置条件

1. 已创建账户类型：硬件钱包、软件钱包、私钥钱包、外部账户、观察账户。  
2. 测试端覆盖：Android / iOS / desktop / ext / web。  
3. 已进入 Market 模块并可打开目标代币详情页。  

---

## 4. 预期结果

1. 有持仓时，详情页正常显示持仓数量和买卖点。  
2. 无持仓时，详情页不展示持仓字段，不展示买卖点。
