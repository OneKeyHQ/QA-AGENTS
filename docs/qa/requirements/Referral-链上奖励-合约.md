# Referral - 链上奖励：合约（On-chain reward: Perps）

> 版本：待确认
> 测试端：iOS / Android / Extension / Desktop / Web
> 创建日期：2026-02-09

---

## 需求背景

实现「链上奖励：合约」详情页，供邀请人查看其邀请的用户所产生的交易数据和返佣奖励。

### 核心改动
- 统一详情页 Header：所有奖励详情页顶部共用统一的筛选控件
- 奖励模块拆分：将原「链上奖励」拆分为「链上奖励：合约」和「链上奖励：DeFi」两个独立模块
- 链上合约逻辑：完善合约返佣的数据展示与筛选逻辑
- 原链上奖励的合约部分代码去掉

---

## 页面结构

- 页面层级：Overview > On-chain reward: Perps
- 奖励模块入口（在「邀请详情 - 奖励」板块中）：
  - 硬件销售奖励（已有功能，本期不改动）
  - 链上奖励：合约（本需求核心页面）
  - 链上奖励：DeFi（后续迭代）

## 功能模块

### 1. 详情页 Header（通用组件）

所有奖励详情页（硬件销售奖励、链上奖励：合约、链上奖励：DeFi）顶部统一使用相同的 Header 组件。

#### 1.1 时间选择器

| 项目 | 规则 |
|------|------|
| 触发按钮 | 显示格式 YYYY/MM/DD - YYYY/MM/DD，右侧带日历图标 |
| 下拉面板 | 双月历并排设计，左侧上月，右侧当月 |
| 导航按钮 | « » 控制年份，‹ › 控制月份 |
| 今日标记 | 当天日期用黄色边框高亮 |
| 选中范围 | 选中的日期范围用黑色背景标识 |
| 未来日期 | 置灰且不可选 |
| 快捷选项 | 今天、本周、昨日、本月、近30天、全部时间（2024-01-01 至今） |
| 默认选择 | 全部时间 |

#### 1.2 筛选按钮（Filter）

| 项目 | 规则 |
|------|------|
| 无筛选状态 | 空心图标（选择 All 时） |
| 有筛选状态 | 实心图标（已选择特定邀请码） |
| 下拉标题 | REFERRAL CODE |
| 选项 | All（默认选中）+ 用户的所有邀请码列表 |
| 选中状态 | 显示 ✓ 标记 |

#### 1.3 导出按钮（Export）

| 项目 | 规则 |
|------|------|
| 触发 | 点击后导出当前筛选条件下的数据 |
| 文件名格式 | {Date1} - {Date2} Onchain Reward - Perps Undistributed / {Date1} - {Date2} Onchain Reward - Perps Total |
| 导出范围 | Undistributed / Total 列表数据（根据用户选择），不隐藏零交易量记录 |
| 导出字段 | Address, Invited Time, Referral Code, Referral Code Remark, First Trade, Volume, Fee, Reward |

### 2. Overview 统计区域

页面顶部展示三个核心指标卡片：

| 指标 | 说明 | 样式 |
|------|------|------|
| Undistributed reward | 筛选条件下尚未发放的奖励总额 | 绿色数字，下方小字显示 Total: $X.XX |
| Volume | 筛选条件下所有交易量总和 | 默认黑色数字 |
| Invited addresses | 筛选条件下的邀请地址总数 | 默认黑色数字，下方小字显示 From X Wallets |

> 重要：Overview 数据始终显示筛选后的全量数据汇总，不受下方 Undistributed/Total Tab 切换影响。

### 3. Details 邀请列表

#### 3.1 Tab 切换逻辑

| Tab | 数据范围 | 特殊控件 |
|-----|---------|---------|
| Undistributed | 仅显示有奖励但尚未发放的记录（rewards > 0） | 无 |
| Total | 显示所有记录（待发放 + 已发放 + 未产生交易） | 显示「Hide zero volume」Switch，默认开启 |

#### 3.2 Hide zero volume 开关

- 仅在 Total Tab 下显示
- 默认状态：开启（隐藏零交易量记录）
- 关闭后：显示所有记录，包括未产生交易的地址

#### 3.3 列表字段定义（桌面端）

| 字段 | 格式 | 排序 | 说明 |
|------|------|------|------|
| ADDRESS | 脱敏显示（如 0x7a23...f8d2） | 不可排序 | 不提供复制功能 |
| INVITATION TIME | YYYY/MM/DD HH:mm | 可排序 ⇅ | 钱包绑定邀请码的时间 |
| REFERRAL CODE | 灰色背景标签样式 | 不可排序 | 该钱包绑定的邀请码 |
| FIRST TRADE | YYYY/MM/DD HH:mm | 可排序 ⇅ | 首次合约交易时间，未交易显示 "--" |
| VOLUME | $X,XXX.XX | 可排序 ⇅ | 选定时间范围内的合约交易总量 |
| FEE | $X.XX | 可排序 ⇅ | 选定时间范围内产生的手续费总额 |
| REWARD | +$X.XX（绿色） / $0.00（灰色） | 可排序 ⇅ | 选定时间范围内为邀请人带来的返佣金额 |

#### 3.4 移动端列表

**收起状态**：
- 左侧：钱包地址（脱敏）
- 右侧：返佣奖励金额（绿色 +$X.XX）
- 副信息行：Vol. $X,XXX.XX | 邀请码标签
- 右侧箭头：点击展开详情

**展开状态**：
- Invitation time: YYYY/MM/DD HH:mm
- Fee: $X.XX
- First Trade: YYYY/MM/DD HH:mm（未交易显示 "--"）

## 空状态处理

### 无邀请记录（新用户）

| 项目 | 内容 |
|------|------|
| 统计卡片 | Undistributed reward: $0.00 / Volume: $0.00 / Invited addresses: 0 |
| 图标 | 礼物盒图标 |
| 标题 | Start Earning Today |
| 描述 | Share your referral code to start earning rewards. |
| 按钮 | Copy & share link（深色实心按钮） |
| 按钮交互 | 点击后复制邀请链接到剪贴板，显示 Toast "Link copied!" |

### 筛选结果为空

| 项目 | 内容 |
|------|------|
| 统计卡片 | 全部显示为 0 |
| 列表区域 | 保留表头，数据区显示空状态 |
| 空状态文案 | No data for selected filters |

---

## 边界情况处理

| 场景 | 处理方式 |
|------|---------|
| First Trade 未发生 | 显示 "--" |
| Volume = 0 | 显示 $0.00 |
| Fee = 0 | 显示 $0.00 |
| Reward = 0 | 灰色文字显示 $0.00 |
| 超长地址 | 统一脱敏：前4位...后4位（如 0x7a...f8d2） |
| 超大金额 | 使用千分位分隔符 |

## 交互逻辑汇总

### 筛选联动

| 操作 | 影响范围 |
|------|---------|
| 时间筛选变更 | 刷新 Overview + 刷新列表 |
| 邀请码筛选变更 | 刷新 Overview + 刷新列表 + Filter 图标变为实心 |
| Tab 切换（Undistributed/Total） | 仅刷新列表，Overview 不变 |
| Hide zero volume 切换 | 仅影响 Total Tab 下的列表显示 |

### 排序逻辑

| 项目 | 规则 |
|------|------|
| 支持排序字段 | Invitation Time, First Trade, Volume, Fee, Reward |
| 排序切换 | 默认 → 升序 → 降序 → 默认 |
| 排序图标 | ⇅（默认）/ ↑（升序）/ ↓（降序） |
| 默认排序 | 按 Volume 降序（最高在前） |

### 移动端交互

| 项目 | 规则 |
|------|------|
| 列表项点击 | 展开/收起详情 |
| 展开限制 | 同时只能展开一个列表项 |
| 下拉刷新 | 支持 |
| 上拉加载更多 | 支持分页加载 |

## 返佣金额计算规则

- 开仓 = 下单仓位金额 × builder fee × 10%
- 平仓 = 下单仓位金额 × builder fee × 10%
- 限价单 = 下单仓位金额 × builder fee × 10%
- 绑定前下单的金额不会被计算到返佣
