# Prime - OneKey ID 信息

> 模块：Prime  
> App 版本：待定  
> 日期：2026-03-20  
> 测试端：iOS / Android / Desktop / Extension / Web

---

## 1. 需求背景

OneKey ID 页面与信息编辑能力整合后，OneKey ID 信息用于在 Prime 体系下承载用户头像、昵称、邮箱与 Prime 状态展示，并支持跨端查看与编辑。OneKey ID 信息既是 OneKey ID 页面核心展示区，也是 Prime 状态与账号能力入口的主要承载位置之一。

---

## 2. 功能描述

### 2.1 菜单与入口展示

| 项目 | 说明 |
|-----|------|
| 未登录态 | 菜单顶部显示 OneKey ID 注册/登录入口与默认头像 |
| 已登录态 | 显示头像、昵称、邮箱、当前登录客户端对应的 Prime 状态图标，并直接展示 Prime/订阅管理（如有）/推荐奖励/已登录设备/我的订单入口 |
| Prime 状态图标 | 绿标表示设备未超限；黄标表示设备超限；无 Prime 不显示 |
| 图标交互（桌面/插件） | 鼠标指向 Prime 状态图标时显示提示层 |
| 图标交互（移动端） | 点击 Prime 状态图标时显示提示层 |
| 图标提示内容 | 提示层展示 Prime 到期时间以及当前设备是否超限；若两种信息都不包含，则不展示提示层 |
| 默认昵称 | 未设置昵称时显示 `OneKey_单词+数字`，超长按 UI 规则省略 |
| 邮箱脱敏 | 按首字母 3 位 + `***` + 后缀显示 |

### 2.2 点击行为与页面结构

| 点击区域 | 结果 |
|---------|------|
| 小屏模式 OneKey ID 个人信息栏 | 进入个人信息编辑页面 |
| 大屏模式「编辑资料」按钮 | 进入个人信息编辑页面 |
| OneKey ID 页面功能区 | 直接展示 Prime / 订阅管理（如有）/ 推荐奖励 / 已登录设备 / 我的订单，不再通过「更多」二级展开 |

### 2.3 OneKey ID 信息编辑

| 项目 | 说明 |
|-----|------|
| 昵称字符 | 不限制字符类型 |
| 昵称长度 | 最多 20 字符，超过无法继续输入 |
| 清空昵称 | 触发不能为空校验 |
| 保存按钮 | 校验通过后可提交；校验报错时不可提交 |
| 网络异常 | 断网状态下提交修改需提示失败 |

### 2.4 头像选择

| 项目 | 说明 |
|-----|------|
| 非 iOS | 打开系统图片选择器 |
| iOS | 首次需申请照片权限 |
| 选择图片 | 支持进入裁剪流程后更新头像预览 |

### 2.5 跨端刷新

| 项目 | 说明 |
|-----|------|
| 资料同步 | 修改成功后，最新头像与昵称同步到其他已登录设备 |
| 状态展示 | 各端按各自当前客户端状态展示 Prime 状态图标；颜色仅表示当前登录客户端自身状态 |

## 3. 业务规则

| 规则项 | 规则描述 |
|-------|---------|
| 登录态区分 | 菜单顶部需区分未登录态与已登录态展示 |
| 图标语义 | Prime 状态图标仅表示当前登录客户端自身状态 |
| 图标交互 | 桌面/插件 hover，移动端点击展示提示层；若无到期时间且当前设备不超限，则不展示提示层 |
| 到期时间展示 | 持续订阅中不显示；赠送或中断订阅显示 |
| 绿标提示门禁 | 订阅中绿标不展示提示 |
| 黄标提示 | 展示设备超限提示（设备数量已达上限） |
| 默认昵称 | 未设置昵称时展示系统生成昵称 |
| 邮箱展示 | 邮箱需按脱敏规则展示 |
| 昵称输入 | 支持多字符类型，最大长度 20 |
| 保存限制 | 昵称校验失败时不可提交 |
| 权限申请 | iOS 修改头像需先处理照片权限 |
| 提交异常 | 网络异常时不得误提示修改成功 |
| 跨端同步 | 头像与昵称修改后需同步到其他登录设备 |
| 入口整合 | 小屏点个人信息栏、大屏点编辑资料按钮进入编辑页 |
| 页面直出 | OneKey ID 页面直出 Prime/订阅管理（如有）/推荐奖励/已登录设备/我的订单 |
| 更多按钮移除 | OneKey ID 页面不再保留「更多」按钮 |
| 删除入口迁移 | 删除 OneKey ID 入口迁移至 OneKey ID 页面删除弹窗 |

## 4. 已知风险

- 特殊字符、Emoji、多语言字符在不同端的显示宽度与截断策略待统一
- iOS 照片权限拒绝后二次申请路径需与系统版本兼容
- 跨端刷新存在缓存时的显示延迟待确认
- Prime 状态图标与设备超限状态的联动依赖 4.5 设备限制规则
