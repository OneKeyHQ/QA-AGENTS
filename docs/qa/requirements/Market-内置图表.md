# Market - 内置图表

> **模块**：Market（行情）
> **功能名称**：Token 详情页内置图表替换
> **版本**：
> **测试端**：Android / iOS / Extension / Desktop / Web
> **关联资源**：现网 TradingView 图表作为回归参考源；`docs/qa/requirements/Market-图表.md` 为通用图表能力需求

---

## 1. 需求背景

Market Token 详情页计划由内置图表替换现有 TradingView 图表。替换后，用户在 Token 详情页看到的价格走势、OHLC、时间周期、技术指标与基础交互保持可用，并在统一标的、统一周期、统一时区下与现网 TradingView 参考口径对齐。

---

## 2. 前置条件

| 条件 | 说明 |
|------|------|
| 入口 | Market 行情列表 / 搜索结果进入 Token 详情页 |
| 参考源 | 同一 Token 使用 TradingView 当前线上图表作为回归参考 |
| 网络 | 默认联网；异常场景单独断网 |
| 账户类型 | 助记词钱包 / 外部账户 / 私钥账户 / 观察账户 / 硬件钱包（按端能力覆盖） |

---

## 3. 功能描述

### 3.1 图表替换范围

| 项 | 说明 |
|----|------|
| 替换对象 | Token 详情页主图表区域由内置图表承接，替换现有 TradingView 渲染 |
| 入口 | 进入 Token 详情页后的图表首屏 |
| 目标 | 保持图表查看、周期切换、指标查看、十字光标、缩放平移等核心能力 |

### 3.2 数据展示

| 项 | 说明 |
|----|------|
| 时间区间 | 支持 1m / 15m / 1h / 4h / D / W 等现有周期能力（以产品最终列表为准） |
| OHLC | O/H/L/C 随周期切换变化；同一根已收盘 K 线可与 TradingView 参考源对照 |
| 价格 / 市值 | 支持价格与市值口径切换（若本版保留该入口） |
| 成交量 | 支持 Volume 展示；与所选周期一致 |
| 次级法币 | 延续 Market 统一 USD 主价格口径；若存在次级法币展示，遵循现有规则 |

### 3.3 技术指标

| 项 | 说明 |
|----|------|
| 指标范围 | 支持 VOL、EMA、MACD、RSI、布林带等现有能力（以产品最终列表为准） |
| 指标显示 | 指标曲线 / 柱体 / 图例与当前周期联动 |
| 多指标组合 | 同时显示多个指标，不互相遮挡到不可读 |
| 指标持久化 | 若产品保留自定义指标能力，切换 Token 后延续相同配置 |

### 3.4 基础交互

| 项 | 说明 |
|----|------|
| 首次加载 | 图表首屏可显示 loading，完成后进入完整图表态 |
| 切换周期 | 切换后图表与 OHLC 同步刷新 |
| 缩放与平移 | 支持查看历史数据 |
| 十字光标 | 支持查看时间、O、H、L、C、Volume 等字段（以端上 UI 为准） |
| 移动端点击 | 若移动端保留点击 K 线详情，点击后展示相应明细 |

### 3.5 对照与异常

| 项 | 说明 |
|----|------|
| 对照方式 | 与 TradingView 对照时统一标的、周期、时区，只比较已收盘 K 线 |
| 实时刷新 | 最新行情更新后，图表数据在合理时间内更新 |
| 空态 / 错态 | 无数据、接口失败、断网时显示空态或错误态，不显示白屏 |
| 长历史 | 加载较长时间范围时仍可交互 |

---

## 4. 业务规则

| 规则项 | 描述 |
|--------|------|
| 对照口径 | 与 TradingView 对照时，优先比较已收盘 K 线；未收盘 K 线不作为主验收对象 |
| 容差策略 | 与参考源对照允许存在小幅行情误差，需在测试记录中注明容差口径 |
| 数据一致性 | 图表收盘价与 Token 详情页主价格、列表价格在产品约定口径下一致 |
| 替换范围 | 图表替换不影响 Token 详情页其他基础信息、买卖入口与外链能力 |
| 回归范围 | 替换图表后，需同步回归法币计价、股票代币、搜索跳详情、跨端渲染一致性 |

---

## 5. 已知风险

| 风险点 | 说明 |
|--------|------|
| 周期默认值 | 默认周期若与现网 TradingView 不同，需单独标注为产品差异而非图表错误 |
| 数据源差异 | 内置图表与 TradingView 可能来自不同聚合源，对照前必须统一交易对、周期与时区 |
| 指标算法差异 | EMA / RSI / MACD / 布林带的初始化与取整口径若不同，需先确认算法定义 |
| 动态区域 | 当前价线、未收盘 K 线、hover 态属于高波动区域，不适合做严格静态截图对照 |
