# Prime - OneKey ID 注册登录

> 模块：Prime  
> App 版本：待定  
> 日期：2026-03-20  
> 测试端：iOS / Android / Desktop / Extension / Web

---

## 1. 需求背景

OneKey ID 是 Prime 会员能力的账号基础。用户需要通过邮箱注册或登录 OneKey ID，完成验证码校验后，才能在当前客户端承载 Prime 状态、会员权益与关联功能。

---

## 2. 功能描述

### 2.1 邮箱输入与校验

| 项目 | 说明 |
|-----|------|
| 正确格式 | 正确邮箱格式时允许继续 |
| 错误格式 | 错误格式提示邮箱地址不正确 |
| 邮箱长度 | 支持 6 至 74 字符 |
| 域名支持 | 支持中文域名 |
| 非法格式示例 | `user@domain`、包含 Emoji 的邮箱均视为非法 |

### 2.2 验证码页

| 项目 | 说明 |
|-----|------|
| 邮箱回显 | 验证码页展示输入邮箱，与邮箱输入页保持一致 |
| 倒计时 | 显示 60 秒倒计时 |
| 正确验证码 | 输入后自动提交 |
| 错误验证码 | 提示验证码无效 |
| 超时验证码 | 正确验证码一般在 2 小时内有效 |
| 重发 | 60 秒后可重新发送验证码 |

### 2.3 登录行为

| 项目 | 说明 |
|-----|------|
| 多端登录 | 同一邮箱在不同端登录当前不限制 |
| 重复登录 | 退出后支持重复接收验证码并重新登录 |
| 成功后状态 | 登录成功后进入 OneKey ID 体系并承载后续 Prime 状态 |

---

## 3. 业务规则

| 规则项 | 规则描述 |
|-------|---------|
| 账号基础 | Prime 相关功能以 OneKey ID 登录态为基础 |
| 邮箱格式校验 | 错误邮箱格式不得进入验证码流程 |
| 长度限制 | 邮箱长度需在 6 至 74 字符范围内 |
| 中文域名 | 中文域名属于有效邮箱类型 |
| 验证码时效 | 验证码页需展示 60 秒倒计时，倒计时结束后允许重发 |
| 自动提交 | 正确验证码输入后自动提交 |
| 多端策略 | 同一邮箱多端登录当前不做限制 |
| 重复进入 | 退出后可重复执行登录流程 |

---

## 4. 已知风险

- 不同邮箱服务商的验证码投递延迟与拦截策略存在差异
- 中文域名邮箱在不同平台输入法与客户端校验上的兼容性待确认
- 验证码有效期与服务端真实过期时间需保持一致
