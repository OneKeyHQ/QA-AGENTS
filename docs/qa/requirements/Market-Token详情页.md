# Market - Token 详情页需求文档

> **模块**：Market（行情）
> **功能名称**：Token 详情页（不含图表与买卖交易）
> **版本**：
> **测试端**：iOS / Android / Extension / Desktop / Web

---

## 1. 需求背景

用户在 Market 行情列表或搜索结果中点击某个 Token 后进入 Token 详情页，用于查看该 Token 的基础信息、关键指标、简介与外部资源链接，并支持分享。

---

## 2. 功能入口

| 入口路径 | 说明 |
|---------|------|
| Market 行情列表 → Token 行 | 点击 Token 行进入详情页 |
| Market 搜索结果 → Token 行 | Symbol/合约地址搜索结果点击 Token 行进入详情页 |

---

## 3. 功能描述

### 3.1 基础信息展示

| 字段/元素 | 说明 |
|---------|------|
| Token 名称/符号 | 展示 Token 名称与 Symbol |
| 网络标识 | 展示所属网络（如 Ethereum、BNB Chain、Solana 等） |
| 合约地址 | EVM 链展示合约地址；展示截断样式并支持复制完整地址 |

### 3.2 核心指标展示

| 字段/元素 | 说明 |
|---------|------|
| 价格 | 展示当前价格（法币计价） |
| 涨跌幅 | 展示 24h 涨跌幅（百分比，颜色区分） |
| 市值 | 展示市值（法币计价，格式化展示） |
| 流通量/总量 | 展示流通量与总量（如适用） |
| 持币地址 | 展示持币地址数（如适用） |

### 3.3 简介与外部资源

| 字段/元素 | 说明 |
|---------|------|
| Token 简介 | 展示项目简介；长文本需可完整查看（如展开/滚动） |
| 官网/社交链接 | 展示官网、Twitter、Telegram 等入口（如有） |

### 3.4 分享

| 字段/元素 | 说明 |
|---------|------|
| 分享入口 | 支持分享 Token 信息或详情页链接 |

---

## 4. 业务规则

| 规则项 | 规则描述 |
|------|---------|
| 数据一致性 | 详情页展示的数据与行情列表/搜索结果中的数据一致（同一 Token、同一网络口径） |
| 外链安全 | 外部链接打开行为可预期（如新窗口/系统浏览器），不应造成页面崩溃或卡死 |
| 地址复制 | 复制后可直接粘贴使用，且与页面展示的 Token 一致 |

---

## 5. 已知风险

| 风险点 | 说明 |
|------|------|
| 多链同名 Token | 同名 Symbol 在不同链存在多个 Token，需确保网络标识与合约地址匹配 |
| 数据源波动 | 价格/市值等字段可能实时变化，需避免闪烁或显示异常格式 |
| 外链不可用 | 官网/社交链接可能 404 或被拦截，需具备可恢复提示路径 |
