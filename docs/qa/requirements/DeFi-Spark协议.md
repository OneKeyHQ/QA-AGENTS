# DeFi - Spark 协议（简单赚币新渠道）

> **App 版本**：<br>
> **测试端**：默认全端（本需求截图基于桌面端）<br>
> **需求来源**：DeFi 新增 Spark 渠道（2026-07-20 用户提供 + 桌面端截图）

## 1. 渠道概述

- **协议**：Spark — 链上资本配置与储蓄平台，将稳定币（USDC、USDT、PYUSD、USDS）和 ETH 部署到 DeFi、CeFi 及现实世界资产中产生收益；由 MakerDAO（现为 Sky）孵化，Phoenix Labs 构建（标签：savings / lending / money-market / stablecoin / defi）
- **网络**：ETH（Ethereum）
- **接入代币**：USDC、USDT（各自独立的认购/赎回入口）
- **金库 / 凭证代币**：spUSDC / spUSDT（详情页「金库」与「凭证代币」两处展示，可跳转区块浏览器）
- **金库管理员**：Spark（可跳转外链）
- **协议信息**：成立时间 2023，团队成员 Sam MacPherson 等，审计 ChainSecurity 等，外链：网站 / X / Discord
- **钱包支持**：HD 与 HW 钱包均支持认购/赎回签名（HW 走设备确认）
- **收益口径**：APY（详情页显示 高 / 低 区间、走势图、收益表现 最近一天/一周/一月）

## 2. 认购规则

| 项 | 规则 |
|---|------|
| 最小认购金额 | 按代币精度（无固定金额门槛，最小一个精度单位即可认购） |
| 最大认购金额 | 无上限（仅受钱包余额限制） |
| 认购金额精度 | 6 位小数（USDC 与 USDT 相同） |
| 认购流程 | 两步：1. 授权 → 2. 认购（底部 Stepper 显示「1. 授权 → 2. 认购」） |
| 预估年收益 | `预估年收益 = APY × 认购金额`（同币种计价 + USD 估值） |
| 余额不足 | 显示「USDC/USDT 不足？」引导，提供「交易」「购买」按钮 |

## 3. 赎回规则（5M 阈值自动判定，用户不可选）

用户**不选择**赎回方式；系统按赎回输入金额与 5M 阈值自动判定，**无提现选项弹窗、无手续费**：

| 赎回方式 | 触发条件 | 页面提示 | 到账时间 |
|---------|---------|---------|---------|
| 立即赎回 | 输入金额 ≤ 5,000,000 | 无任何提示 | 立即到账 |
| 排队赎回 | 输入金额 > 5,000,000 | 「需要几分钟才能赎回到账」 | 几分钟 |

- **赎回金额范围**：最小按代币精度（无固定金额门槛）；最大为当前持仓（可 100% 全部赎回）
- **赎回无手续费**（已确认）：立即赎回与排队赎回到账金额均 = 赎回金额
- **5M 阈值为后台可配置参数**：测试时可由后台调小阈值，用小额真实提交验证排队赎回路径；生产默认 5,000,000
- 阈值判定对用户透明：金额一旦超过上限即自动走排队赎回流程，页面仅出现到账时间提示，不出现方式选择入口

## 4. 持仓

- 详情页顶部提供「持仓」入口；持仓列表按渠道分组显示 Spark 组
- 持仓行提供「管理」按钮 → 打开认购/赎回管理弹窗（含「历史记录」入口）
