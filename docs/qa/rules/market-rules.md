# Market 模块测试规则文档

> 本文档记录 Market 模块的核心测试规则。
> 生成 Market 模块测试用例时，必须参考本文档。

---

## 1. 模块概述

Market 模块包含以下核心功能：
- 行情列表（Token 价格、涨跌幅）
- K 线图表（价格走势、技术指标）
- Token 详情页
- Swap Pro Mode（快捷兑换）
- 限价单功能

---

## 2. 行情列表规则

### 2.1 数据展示

| 字段 | 规则 |
|-----|------|
| Token 名称 | 显示名称和符号 |
| 价格 | 显示当前价格（法币） |
| 涨跌幅 | 显示 24h 涨跌幅（百分比） |
| 成交量 | 显示 24h 成交量 |
| 市值 | 显示市值（可选） |

### 2.2 列表功能

| 功能 | 规则 |
|-----|------|
| 排序 | 支持按价格、涨跌幅、成交量排序 |
| 搜索 | 支持按名称、符号搜索 |
| 收藏 | 支持收藏 Token，显示在收藏列表 |
| 分类 | 支持按分类筛选（热门、DeFi、NFT 等） |

### 2.3 数据更新

| 规则项 | 规则描述 |
|-------|---------|
| 实时更新 | WebSocket 推送价格更新 |
| 刷新频率 | 价格更新节流（防止闪烁） |
| 离线处理 | 断网时显示缓存数据 + 离线提示 |

---

## 3. K 线图表规则

### 3.1 图表功能

| 功能 | 规则 |
|-----|------|
| 时间周期 | 支持 1m/5m/15m/1h/4h/1d/1w 等 |
| 缩放 | 支持手势缩放、拖动 |
| 十字线 | 显示价格、时间详情 |
| 指标 | 支持添加技术指标（MA、MACD、RSI 等） |

### 3.2 数据一致性

| 规则项 | 规则描述 |
|-------|---------|
| K 线/列表一致 | K 线收盘价与列表价格一致 |
| 多源策略 | 取中位/优先源/降级 |
| OHLC 数据 | Open/High/Low/Close 数据正确 |

### 3.3 性能要求

| 指标 | 要求 |
|-----|------|
| 加载时间 | K 线数据加载 < 2s |
| 帧率 | 交互帧率 ≥ 30 FPS |
| 内存 | 长时间查看不出现内存泄漏 |

---

## 4. Token 详情页规则

### 4.1 信息展示

| 字段 | 规则 |
|-----|------|
| 价格 | 当前价格 + 涨跌幅 |
| 市值 | 市值排名 |
| 流通量 | 流通量/总量 |
| 简介 | Token 项目简介 |
| 官网/社交 | 官网、Twitter、Telegram 等链接 |

### 4.2 操作入口

| 操作 | 规则 |
|-----|------|
| 买入/卖出 | 跳转 Swap 或 Swap Pro Mode |
| 收藏 | 添加到收藏列表 |
| 分享 | 分享 Token 信息 |
| 提醒 | 设置价格提醒（可选） |

## 5. Swap Pro Mode 规则

> Swap Pro Mode 是 Market 模块下的快捷兑换功能，入口在 Token 详情页。

### 5.1 功能概述

| 规则项 | 规则描述 |
|-------|---------|
| 入口 | Market Token 详情页 |
| 状态机 | 询价→路径→授权→Swap→Pending→Success/Fail |

### 5.2 核心验证

| 验证项 | 规则 |
|-------|------|
| 滑点设置 | 支持自定义滑点 |
| Gas 费用 | 显示预估 Gas 费 |
| 价格影响 | 显示 Price Impact |
| 最小收到量 | 显示 Minimum Received |
| 授权流程 | ERC20 代币需要授权 |

### 5.3 与普通 Swap 差异

| 差异点 | Pro Mode | 普通 Swap |
|-------|---------|----------|
| 入口 | Token 详情页 | Swap 页面 |
| 默认 Token | 当前查看的 Token | 需手动选择 |
| 行情数据 | 使用 Market 行情 | 使用 Swap 报价 |

## 6. 限价单规则

> 限价单功能通过 CowSwap 渠道实现，支持网络：Ethereum、Arbitrum、Base

### 6.1 基本规则

| 规则项 | 规则描述 |
|-------|---------|
| FromToken | 不支持直接出售 ETH，需换成 WETH |
| ETH → WETH | 自动转换逻辑验证 |

### 6.2 订单有效期

| 选项 | 说明 |
|-----|------|
| 5 分钟 | 短期订单 |
| 30 分钟 | 短期订单 |
| 1 小时 | 中期订单 |
| 1 天 | 中期订单 |
| 3 天 | 长期订单 |
| 7 天 | 长期订单 |
| 1 个月 | 长期订单 |

### 6.3 部分成交

| 规则项 | 规则描述 |
|-------|---------|
| 启用 | 允许部分成交 |
| 禁用 | 仅完全成交 |
| 处理逻辑 | 需验证两种模式的不同处理 |

### 6.4 特殊授权规则

| 场景 | 规则 |
|-----|------|
| USDT → ETH | Ethereum 上需要二次授权 |
| USDT → 其他 | 不需要二次授权 |

## 7. 通用规则

### 数据一致性
- K 线/列表/详情价格一致
- 多源差异策略（取中位/优先源/降级）

### 状态机
- WS 断连 → 重连 → 补数据 → 防闪烁
- 切币/切交易对/切法币

### 极端行情
- 精度丢失处理
- 科学计数法显示
- 涨跌幅异常处理
- 停牌/新币无数据处理

### 性能
- 长列表虚拟化
- K 线交互帧率
- 缓存/预加载策略

### 可观测
- 数据源降级提示
- 请求失败可恢复路径

## 8. 规则维护指南

### 如何更新规则

1. **发现规则变更**：
   - 在测试过程中发现规则与文档不一致
   - 收到产品/开发通知规则变更
   - 限价单支持新网络

2. **更新文档**：
   - 直接修改对应章节
   - 在变更记录中记录更新时间和原因
