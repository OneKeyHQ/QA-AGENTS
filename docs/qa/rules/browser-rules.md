# Browser 模块测试规则文档

> 本文档记录 Browser（DApp）模块的核心测试规则。
> 生成 Browser/DApp 相关测试用例时，必须参考本文档。

---

## 1. 模块概述

Browser 模块包含以下核心功能：
- DApp 浏览器
- 钱包连接（WalletConnect、Injected）
- 签名请求处理
- 交易确认
- 权限管理

---

## 2. DApp 连接规则

### 2.1 连接方式

| 连接方式 | 说明 | 适用场景 |
|---------|-----|---------|
| Injected | 注入式连接 | Extension、Desktop |
| WalletConnect | 扫码连接 | Mobile、跨设备 |
| EIP-6963 | 多钱包发现 | Extension |

### 2.2 连接状态机

```
Disconnected → Connecting → Connected → Disconnected
                  ↓
              Failed (超时/拒绝)
```

### 2.3 连接规则

| 规则项 | 规则描述 |
|-------|---------|
| Origin 展示 | 显示 DApp 域名 |
| 权限请求 | 显示请求的权限范围 |
| 账户选择 | 支持选择连接的账户 |
| 网络选择 | 支持选择连接的网络 |

---

## 3. 签名请求规则

### 3.1 签名类型

| 签名类型 | 说明 | 风险等级 |
|---------|-----|---------|
| personal_sign | 个人签名 | 低 |
| eth_signTypedData | 类型化数据签名 | 中 |
| eth_sign | 原始签名 | 高（危险） |
| eth_sendTransaction | 交易签名 | 高 |

### 3.2 签名内容解析

| 规则项 | 规则描述 |
|-------|---------|
| 内容解析 | 尽可能解析签名内容 |
| 可读展示 | 以可读格式展示（非十六进制） |
| 风险提示 | 高风险操作显示警告 |

### 3.3 盲签处理

| 场景 | 规则 |
|-----|------|
| 解析成功 | 显示解析后的内容 |
| 解析失败 | 显示"无法解析"警告 |
| 盲签阻断 | 高危操作强制警告，需二次确认 |

---

## 4. 交易确认规则

### 4.1 交易详情展示

| 字段 | 规则 |
|-----|------|
| From | 发送方地址 |
| To | 接收方地址/合约地址 |
| Value | 转账金额 |
| Data | 合约调用数据（可折叠） |
| Gas | Gas Limit 和 Gas Price |
| 总费用 | 预估总费用（法币） |

### 4.2 合约交互解析

| 规则项 | 规则描述 |
|-------|---------|
| 方法识别 | 识别常见合约方法（transfer、approve 等） |
| 参数解析 | 解析并展示方法参数 |
| 金额展示 | 转账金额、授权额度等 |

### 4.3 风险提示

| 风险类型 | 规则 |
|---------|------|
| 高额授权 | 授权额度过大时警告 |
| 未知合约 | 未验证合约时提示 |
| 钓鱼检测 | 相似域名/已知钓鱼地址警告 |

---

## 5. 权限管理规则

### 5.1 已连接 DApp

| 功能 | 规则 |
|-----|------|
| 列表展示 | 显示所有已连接的 DApp |
| 详情查看 | 显示连接时间、权限范围 |
| 断开连接 | 支持断开单个/所有连接 |

### 5.2 权限范围

| 权限 | 说明 |
|-----|------|
| 查看地址 | 读取账户地址 |
| 查看余额 | 读取账户余额 |
| 请求签名 | 发起签名请求 |
| 请求交易 | 发起交易请求 |

---

## 6. 安全规则

### 6.1 钓鱼检测

| 检测项 | 规则 |
|-------|------|
| 域名检测 | 检测相似域名（如 uniswap vs un1swap） |
| 黑名单 | 已知钓鱼地址/域名拦截 |
| 新域名 | 首次访问的域名提示 |

### 6.2 Origin 验证

| 规则项 | 规则描述 |
|-------|---------|
| 展示 | 始终显示请求来源域名 |
| 对比 | 与已知安全域名对比 |
| 警告 | 可疑域名显示警告 |

### 6.3 盲签阻断

| 场景 | 处理方式 |
|-----|---------|
| 无法解析 | 显示强警告 |
| eth_sign | 默认禁用，需手动开启 |
| 高危操作 | 需要二次确认 |

---

## 7. 兼容性规则

### 7.1 多 Tab 处理

| 场景 | 规则 |
|-----|------|
| 多 Tab 请求 | 按时间顺序处理 |
| Tab 切换 | 保持连接状态 |
| Tab 关闭 | 断开对应连接（可配置） |

### 7.2 EIP-6963 支持

| 规则项 | 规则描述 |
|-------|---------|
| 钱包发现 | 支持多钱包发现协议 |
| 优先级 | 用户选择优先钱包 |
| 冲突处理 | 多钱包冲突时提示选择 |

### 7.3 网络切换

| 场景 | 规则 |
|-----|------|
| DApp 请求切换 | 弹窗确认 |
| 用户主动切换 | 通知 DApp 网络变更 |
| 不支持网络 | 提示添加网络 |

---

## 8. 通用规则

### 状态机
- connect → 请求权限 → 签名/交易 → 用户确认/拒绝 → 回到 DApp 回传

### 安全
- origin 展示
- 权限范围
- 钓鱼/相似域名
- 盲签阻断（解析失败时强警示）

### 兼容
- 多 tab
- EIP-6963
- 多钱包冲突
- 网络切换请求处理

### 链上不确定性
- 签名弹窗中断
- 返回后恢复
- 重复请求幂等

### 可观测
- 签名内容解析结果可验证
- method/params/金额/收款地址/合约地址

## 9. 规则维护指南

### 如何更新规则

1. **发现规则变更**：
   - 在测试过程中发现规则与文档不一致
   - 收到产品/开发通知规则变更
   - 新的 EIP 标准支持

2. **更新文档**：
   - 直接修改对应章节
   - 在变更记录中记录更新时间和原因
