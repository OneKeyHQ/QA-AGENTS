# Prime 模块测试规则文档

> 本文档记录 Prime 模块的核心测试规则。
> 生成 Prime 相关测试用例时，必须参考本文档。

---

## 1. 模块概述

Prime 模块包含以下核心功能：
- 会员权益展示
- 订阅/续费流程
- 会员专属功能
- 订阅管理

## 4.3 OneKey Cloud 规则

### 开启条件

| 规则项 | 规则描述 |
|-------|---------|
| 自动锁定 | 无法使用「永不」和「离开 4 小时」，需改为 4 小时以下才能开启 |
| 首次开启 | 需设置 OneKeyID 密码 |
| 非首次开启 | 需验证 OneKeyID 密码 |

### 密码规则

| 规则项 | 规则描述 |
|-------|---------|
| 检查条件 | 12 字符、数字、字母、英文特殊字符 |
| 复杂度 | 弱/良/强（示例：ABCDEFG123456& 为弱，#abC20260101 为良,qazPLM$14700 为强） |
| 提交条件 | 满足所有条件、复杂度不为弱、两次输入一致 |
| 可见/复制 | 密码输入框可切换可见，可见状态下允许复制 |

### 同步规则

| 同步类型 | 规则 |
|---------|------|
| HD/HW/QR/隐藏钱包账户名 | 修改同步到其他端，旧名称加入设备历史 |
| 私钥/公钥/观察钱包账户名 | 仅同步对应账户 |
| 空passphrase隐藏钱包 与标准钱包 | 名称分别同步 |
| 自定义代币/网络/RPC、浏览器书签、市场观察列表、地址簿 | 新增/修改同步 |
| 异步触发 | 打开账户选择器/地址簿等，5 分钟间隔；云同步开关/断线重连/手动同步无间隔 |

### 冲突规则

| 场景 | 结果 |
|-----|------|
| 关闭同步开关，账户名称：A 先改 A2，B 后改 B2 | 开启同步开关，B2 生效 |
| 关闭同步开关，通讯地址名称：B 先改 B3，A 后改 A3 | 开启同步开关，A3 生效 |

### 密码变更

| 规则项 | 规则描述 |
|-------|---------|
| 修改密码 | 需验证 APP 密码和 OneKeyID 密码 |
| B 端效果 | 提示密码已被修改或重置，自动关闭云同步开关 |
| 忘记密码 | 需勾选「我明白这将永久删除我的云备份数据」才能删除 |

## 4.4 OneKey ID 个人资料规则

| 规则项 | 规则描述 |
|-------|---------|
| 菜单显示 | 头像、昵称、邮箱、Prime 状态图标（正常绿标/超限黄标） |
| 默认昵称 | 未设置时为 OneKey_单词+数字，超长省略 |
| 邮箱脱敏 | 首字母 3 位+***+后缀 |
| 昵称限制 | 不限制字符类型，最多 20 字符，超过无法输入，清空提示不能为空 |
| 头像选择 | 非 iOS 打开系统选择器；iOS 需申请照片权限 |
| 网络异常 | 断开网络修改提示失败 |

## 4.5 Prime 登陆设备限制

| 规则项 | 规则描述 |
|-------|---------|
| 设备数量 | 前 5 台显示绿色标签，第 6 台起显示黄色超限标签 |
| 设备列表 | 按最后活跃时间排序 |
| 移除超限设备 | 当前设备状态不变 |
| 移除未超限设备 | 当前设备变为未超限 |
| 未超限设备登出 | 最后活跃的超限设备变为未超限 |

## 4.6 OneKey ID 注册/登录规则

| 规则项 | 规则描述 |
|-------|---------|
| 邮箱格式 | 正确格式可继续；错误提示邮箱地址不正确 |
| 邮箱长度 | 6~74 字符；支持中文域名 |
| 非法格式 | user@domain、emoji 提示不正确 |
| 验证码 | 60s 倒计时；错误提示无效；正确自动提交；超时验证码一般 2 小时内有效 |
| 重发 | 60s 后可重新发送 |

## 4.7 批量复制地址规则

### 支持范围

| 支持 | 不支持 |
|-----|--------|
| 软件钱包、无私钥钱包、硬件钱包、隐藏钱包 | 二维码钱包、私钥钱包、观察钱包、外部账户、未备份软件钱包、已重置硬件钱包 |

### 入口

| 入口 | 规则 |
|-----|------|
| 首页更多、Prime 权益页 | 唤起弹窗，默认选择当前钱包 |
| 账户管理器 | 仅软件/硬件/隐藏钱包有入口 |

### 弹窗规则

| 规则项 | 规则描述 |
|-------|---------|
| 网络默认 | 首页网络；All networks 下默认 Bitcoin |
| 派生路径 | 默认与设置页一致；TON 为 v4R2；其他为 BIP44 |
| 从 | 默认 1，最大 2147483548 |
| 生成数量 | 默认 10，最大 100 |
| 导出 CSV | 文件名：钱包名称_网络名称_addresses_时间戳.csv；列：Account name、地址、Derivation path |
| 硬件无设备导出 | 可导出，显示「地址未验证」红色标签 |

## 4.8 批量撤销授权规则

### 支持范围

| 支持 | 不支持 |
|-----|--------|
| 软件钱包、无私钥钱包、私钥账户、外部账户、硬件钱包、QRWallet | 观察账户 |

### 功能

| 模式 | 规则 |
|-----|------|
| 逐一取消 | 逐个签名提交 |
| 批量撤销 | Prime 专属；软件钱包自动批量；硬件钱包需逐一签名 |
| 支持网络 | ethereum, bsc, polygon, arbitrum, avalanche, optimism, base |

---

## 4.9 提升交易通知账户上限规则

| 规则项 | 规则描述 |
|-------|---------|
| 默认上限 | 20 个账户 |
| Prime 生效 | 开启 Prime 后上限变更为 100 个账户 |
| Prime 失效/登出 | Prime 失效或登出 OneKey ID 后，上限恢复为 20 个 |
| 超限关闭 | 权限生效期间开启超过 20 个账户通知时，Prime 失效后按开启时间自动关闭后开启的账户通知，保留前 20 个 |
| 再次开启 | 再次开启 Prime 后，自动恢复此前被关闭的账户通知 |

---

## 4.10 OneKey ID 删除规则

| 规则项 | 规则描述 |
|-------|---------|
| 前置 | 需勾选 checkbox 才能继续；未登录不显示删除选项 |
| 验证 | 需输入 APP 密码；向注册邮箱发送验证码 |
| 删除后 | 其他端数据同步停止、自动退出；邀请码无效；已绑定钱包保持绑定 |
| 重新注册 | 同邮箱可重新注册，生成新用户，不继承删除前数据 |

## 6. 支付规则

### 6.1 应用内购买

| 平台 | 规则 |
|-----|------|
| iOS | 唤起 App Store 支付 |
| Android（支持 Google 框架，商店安装） | 唤起 Google Play 支付 |
| Android（支持 Google 框架，APK 安装） | 弹窗选择 Google Play 或 webview 支付 |
| Android（不支持 Google 框架） | 唤起 webview 支付 |
| 其他端 | 唤起 webview 支付 |

### 6.1.1 管理订阅

| 支付渠道 | 操作设备 | 预期 |
|---------|---------|------|
| iOS/Google 商店 | 支持对应商店的设备 | 打开对应商店订阅管理 |
| iOS/Google 商店 | 不支持对应商店的设备 | 通过网页打开对应商店订阅管理 |
| webview | 任意设备 | 发送管理链接邮件到登陆邮箱 |

### 6.1.2 套餐修改（仅 iOS）

| 操作 | 规则 |
|-----|------|
| 月费转年费 | 直接切换 |
| 年费转月费 | 从到期时间开始变更，当期到期后切换 |

### 6.2 加密货币支付

| 规则项 | 规则描述 |
|-------|---------|
| 支持币种 | 支持的支付币种列表 |
| 支付流程 | 生成支付地址 → 转账 → 确认 → 激活 |
| 超时处理 | 支付超时处理 |

### 6.3 兑换码

| 规则项 | 规则描述 |
|-------|---------|
| 格式 | 兑换码格式规范 |
| 验证 | 兑换码有效性验证 |
| 激活 | 兑换成功后自动激活 |
| 重复使用 | 兑换码只能使用一次 |

### 如何更新规则

1. **发现规则变更**：
   - 在测试过程中发现规则与文档不一致
   - 收到产品/开发通知规则变更

## 9. 订单管理规则

> 详见需求文档：`docs/qa/requirements/Prime-订单管理.md`

### 9.1 功能入口

| 入口 | 路径 |
|-----|------|
| 订单列表 | OneKey ID 弹窗 → My orders |

### 9.2 邮箱匹配规则

| 规则项 | 规则描述 |
|-------|---------|
| 匹配条件 | 官网下单邮箱 = OneKey ID 邮箱 |
| 匹配成功 | 显示该邮箱关联的所有订单 |
| 匹配失败 | 显示空状态 |

### 9.3 状态场景

| 场景 | 显示内容 |
|-----|---------|
| 无订单 | 空状态图 + "No orders yet" + Find my order / Buy OneKey 按钮 |
| 有订单 | 订单列表（订单号、数量、日期、状态、金额）+ Details 按钮 |

### 9.4 订单状态类型

订单状态仅包含以下四种：

| 状态 | 说明 |
|-----|------|
| 待发货 | 订单未发货 |
| 已发货 | 已发货 |
| 已取消 | 已取消 |
| 已收货 | 用户已收货 |

### 9.5 跳转规则

| 操作 | 目标 |
|-----|------|
| Find my order | orders.onekey.so |
| Buy OneKey | shop.onekey.so |
| Details | 官网订单详情页（含物流信息） |

### 9.6 加载规则

| 规则项 | 规则描述 |
|-------|---------|
| 加载方式 | 一次性加载 |
| 分页 | 暂不支持 |
