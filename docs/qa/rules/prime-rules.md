# Prime 模块测试规则文档

> 本文档记录 Prime 模块的核心测试规则。
> 生成 Prime 相关测试用例时，必须参考本文档。

---

## 1. 模块概述

Prime 模块包含以下核心功能：
- 会员权益展示
- 订阅/续费流程
- 会员专属功能
- 订阅管理

---

## 2. 会员权益规则

### 2.1 权益展示

| 规则项 | 规则描述 |
|-------|---------|
| 权益列表 | 展示所有 Prime 会员权益 |
| 对比展示 | 普通用户 vs Prime 用户权益对比 |
| 权益说明 | 每个权益的详细说明 |

### 2.2 权益类型

| 权益类型 | 说明 |
|---------|-----|
| 功能解锁 | 会员专属功能 |
| 费用优惠 | 交易手续费减免 |
| 服务优先 | 客服优先响应 |
| 专属活动 | 会员专属活动参与资格 |

---

## 3. 订阅流程规则

### 3.1 订阅入口

| 入口位置 | 规则 |
|---------|------|
| 设置页 | Prime 会员入口 |
| 权益引导 | 使用受限功能时引导订阅 |
| 活动推广 | 活动页面订阅入口 |

### 3.2 订阅方式

| 方式 | 规则 |
|-----|------|
| 应用内购买 | iOS App Store / Google Play |
| 加密货币 | 支持加密货币支付 |
| 兑换码 | 支持兑换码激活 |

### 3.3 订阅周期

| 周期 | 规则 |
|-----|------|
| 月付 | 按月订阅 |
| 年付 | 按年订阅（优惠价格） |
| 终身 | 一次性购买（如有） |

### 3.4 订阅状态机

```
未订阅 → 订阅中 → 已订阅 → 到期 → 续费/过期
                      ↓
                  取消订阅
```

## 5. 订阅管理规则

## 7. 通用规则

### 状态同步
- 订阅状态跨端同步
- 多设备状态一致

### 异常处理
- 支付失败处理
- 网络异常处理
- 订阅状态异常恢复

### 可观测
- 订阅成功/失败原因可追溯
- 支付记录可查询

## 8. 规则维护指南

### 如何更新规则

1. **发现规则变更**：
   - 在测试过程中发现规则与文档不一致
   - 收到产品/开发通知规则变更
   - 新增会员权益

2. **更新文档**：
   - 直接修改对应章节
   - 在变更记录中记录更新时间和原因

## 9. 订单管理规则

> 详见需求文档：`docs/qa/requirements/Prime-订单管理.md`

### 9.1 功能入口

| 入口 | 路径 |
|-----|------|
| 订单列表 | OneKey ID 弹窗 → My orders |

### 9.2 邮箱匹配规则

| 规则项 | 规则描述 |
|-------|---------|
| 匹配条件 | 官网下单邮箱 = OneKey ID 邮箱 |
| 匹配成功 | 显示该邮箱关联的所有订单 |
| 匹配失败 | 显示空状态 |

### 9.3 状态场景

| 场景 | 显示内容 |
|-----|---------|
| 无订单 | 空状态图 + "No orders yet" + Find my order / Buy OneKey 按钮 |
| 有订单 | 订单列表（订单号、数量、日期、状态、金额）+ Details 按钮 |

### 9.4 订单状态类型

| 状态 | 说明 |
|-----|------|
| Pending | 待处理 |
| Processing | 处理中 |
| Shipped | 已发货 |
| Delivered | 已送达 |
| Cancelled | 已取消 |
| Refunded | 已退款 |

### 9.5 跳转规则

| 操作 | 目标 |
|-----|------|
| Find my order | orders.onekey.so |
| Buy OneKey | shop.onekey.so |
| Details | 官网订单详情页（含物流信息） |

### 9.6 加载规则

| 规则项 | 规则描述 |
|-------|---------|
| 加载方式 | 一次性加载 |
| 分页 | 暂不支持 |
