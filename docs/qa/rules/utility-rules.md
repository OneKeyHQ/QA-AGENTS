# Utility 模块测试规则文档

> 本文档记录通用业务模块的核心测试规则。
> 生成通用功能相关测试用例时，必须参考本文档。

---

## 1. 模块概述

Utility 模块包含以下核心功能：
- 设置（语言、主题、安全等）
- 通知（推送、消息中心）
- 风控（设备检测、交易拦截）
- 地址簿
- 扫码
- 搜索
- App 升级

## 7. 搜索规则

### 7.1 全局搜索

| 搜索范围 | 规则 |
|---------|------|
| Token | 按名称、符号搜索 |
| DApp | 按名称搜索 |
| 地址 | 按地址搜索 |
| 历史记录 | 搜索历史交易 |

### 7.2 搜索体验

| 规则项 | 规则描述 |
|-------|---------|
| 实时搜索 | 输入即搜索（防抖） |
| 高亮匹配 | 匹配文字高亮 |
| 搜索历史 | 显示最近搜索 |
| 热门推荐 | 显示热门搜索 |

### 7.3 通用搜索框（Wallets / Tokens / dApps / My assets / Perps / Settings / All）

#### 7.3.1 Wallets Tab（账户）

- 地址搜索：
  - 输入非已有账户地址：展示 URL 钱包结果
  - 输入已有账户地址：展示已有账户结果
  - 地址不支持模糊搜索
  - 地址区分大小写
- 账户名搜索：
  - 不区分大小写，支持模糊匹配（例如输入 `acco` 可命中包含 `Account` 的账户）
  - 在单链网络下：账户搜索结果展示地址（前八位…后六位）
  - 在 `All networks` / `BTC` / `LTC` 等多链筛选下：账户搜索结果不展示地址
- 搜索结果点击跳转：
  - URL 钱包结果：跳转至对应网络的 URL 钱包页面
  - 已有账户结果：跳转至 wallet 页面，并切换为当前账户与对应网络

#### 7.3.2 Tokens Tab（代币）

- 输入 Token Symbol/关键词：支持精准与模糊搜索，不区分大小写
- 列表展示字段包含：Token 图标、Symbol、名称、实时币价
- 排序规则：按 Tokens 7天总市值排序
- 支持搜索价格为 `null` 的 Token（如 POWER/AIP）
- 点击搜索结果：跳转到对应 Token 的币种详情 modal

#### 7.3.3 dApps Tab（DApp）

- 域名输入（完整/不完整）：展示访问第三方网站选项与谷歌搜索选项，点击后打开新标签页
- 内容关键词输入（精准/模糊，不区分大小写）：
  - 展示谷歌搜索联想项
  - 展示匹配的 dApp 联想项
- 谷歌搜索项点击：在新标签页打开谷歌搜索，并带入输入内容
- dApp 项点击：在新标签页打开对应 dApp
- 移动端发现 tab 场景：在 `Defi` 页面切换后输入 `uniswap`，点击相关选项应跳转至发现 tab 页面并打开新标签页

#### 7.3.4 My assets Tab（我的资产）

- 搜索范围仅面向当前账户
- 不支持观察地址、私钥、三方钱包作为搜索输入
- 资产搜索：
  - 合约地址：匹配成功
  - Symbol：大小写不敏感
  - 名称：大小写不敏感

#### 7.3.5 Perps Tab（合约）

- 合约 Token 搜索：支持精准与模糊搜索，不区分大小写
- 列表展示字段包含：Token 图标、Symbol、名称、Badge、美元价格
- 排序规则：按服务端返回排序
- 点击结果：跳转至对应 Token 的 Perps 详情页
- 不支持 Perps 的 token（如 USDC/REQ）：不返回搜索结果

#### 7.3.6 Settings Tab（设置）

- 通过钱包关键字搜索：展示钱包和 dapp 账户对齐选项、防护管理选项
- 点击“钱包和 dapp 账户对齐”：跳转至对应设置页面

#### 7.3.7 All Tab（聚合）

- 非 Market 上下文下聚合顺序：`Wallets > Tokens > My assets > dApps`
- Market 上下文下聚合顺序：`Tokens > Wallets > My assets > dApps`
- 展示规则：
  - 无结果板块不展示
  - 每个板块最多展示前 5 个结果
  - 超过 5 个结果展示“显示更多”
  - 点击“显示更多”：切换到对应 Tab，并保留输入内容

## 9. Mac 菜单栏（Menu Bar / Tray）规则

### 9.1 平台与开关

| 规则项 | 规则描述 |
|-------|---------|
| 平台支持 | 仅 macOS 完整支持；Windows / Linux 不出现入口、不报错 |
| 设置入口 | `Settings > Preferences > Menu Bar` 开关 |
| 开关行为 | 开 → 立即创建图标；关 → 立即销毁图标并关闭面板 |
| 持久化 | 重启后保持用户上次选择 |
| 升级默认 | 旧版本无该字段历史值时默认开启 |

### 9.2 面板基础行为

| 规则项 | 规则描述 |
|-------|---------|
| 面板尺寸 | 360 × 480 固定 |
| 弹出位置 | 锚定在菜单栏图标下方，多显示器场景跟随触发屏幕 |
| 隐藏触发 | 失焦自动隐藏 / Esc 自动隐藏 / 再次点击图标 toggle |
| 唯一实例 | 同一时刻只允许一个面板 |

### 9.3 面板内容

| 区块 | 内容 |
|-----|------|
| 顶部账户区 | 钱包名称 / 头像 emoji / Account #N / 总资产（USD） |
| 涨跌区 | 24h 涨跌（百分比 + 金额，正绿负红） |
| 自选行情 | 用户在主窗口收藏的现货 / 合约（图标、名称、链、价格、涨跌） |
| 待处理交易 | pending 交易列表（类型、金额、状态徽标） |

### 9.4 空态与异常

| 状态 | 表现 |
|-----|------|
| 无钱包 | 引导文案 + 「打开主窗口创建/导入钱包」按钮 |
| 应用锁定 | 显示 "App is Locked"，停止有效轮询 |
| 无自选 / 无待处理 | 区块内空态文案（暂无自选 / 暂无待处理交易） |
| 网络离线 | 显示离线提示，保留上次缓存数据 |

### 9.5 点击跳转

> 自选按数据源分两条独立路径；待处理交易不跳页，而是在主窗口内打开历史记录弹窗。

| 点击位置 | 数据源 | 行为 |
|---------|-------|-----|
| 顶部账户区 | 当前账户 | 唤起主窗口 → 首页 / 当前账户概览 |
| 自选现货 | Market | 唤起主窗口 → Market 下的代币详情页 |
| 自选合约 | Perps | 唤起主窗口 → Perps 下的合约详情页 |
| 待处理交易 | 当前账户 pending | 唤起主窗口 → 主窗口内打开历史记录弹窗并定位该交易（不跳独立页面） |

### 9.5.1 账户隔离（核心规则）

Tray 数据按当前选中账户严格隔离，切换账户后立即同步刷新：

| 区域 | 规则 |
|-----|------|
| 待处理交易 | 只显示当前账户 pending；切换账户后刷新；不串账户、不残留 |
| 顶部钱包账户与余额 | 钱包名 / 头像 / Account #N / 总资产 / 24h 涨跌均跟随当前账户 |
| 切换中间态 | 不允许"前账户余额 + 新账户交易"混合渲染；最终收敛到新账户 |
| 切换钱包 | HD ↔ HW 等钱包切换同样适用 |

### 9.6 通知与轮询

| 规则项 | 规则描述 |
|-------|---------|
| 通知形式 | macOS 原生系统通知（Notification Center） |
| 触发条件 | pending → confirmed 或 pending → failed |
| 通知点击 | 唤起主窗口 → 定位到该交易 |
| 锁屏期间 | 停止有效轮询，不发"已确认"通知 |
| 解锁恢复 | 差量校准状态，不补发误报通知 |
| 同状态重复回推 | 仅推送 1 次，不重复 |

### 9.7 主题与本地化

| 规则项 | 规则描述 |
|-------|---------|
| 系统主题 | 应用主题为「跟随系统」时实时切换；强制深/浅色时保持 |
| 应用语言 | 切换语言时实时刷新面板文案 |

### 9.8 主窗口安全边界（IPC）

| 来源 | 行为 |
|-----|------|
| 主窗口 | OAuth / HttpServer / Tray 等受控 IPC 按预期响应 |
| 非主窗口（含 Tray、其他 webview） | 拦截非授权调用，不破坏主流程，输出可追踪日志 |

### 9.9 资产总值口径

| 规则项 | 规则描述 |
|-------|---------|
| 计算公式 | 总值 = token 总值 + DeFi 总值 |
| 覆盖分支 | 特定网络 / 全网络 / merge derive |
| 一致性 | Tray 顶部、Home 概览、账户详情三处总值完全相等 |
