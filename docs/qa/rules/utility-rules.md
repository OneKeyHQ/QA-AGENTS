# Utility 模块测试规则文档

> 本文档记录通用业务模块的核心测试规则。
> 生成通用功能相关测试用例时，必须参考本文档。

---

## 1. 模块概述

Utility 模块包含以下核心功能：
- 设置（语言、主题、安全等）
- 通知（推送、消息中心）
- 风控（设备检测、交易拦截）
- 地址簿
- 扫码
- 搜索
- App 升级

---

## 2. 设置模块规则

### 2.1 语言设置

| 规则项 | 规则描述 |
|-------|---------|
| 支持语言 | 中文、英文、日文、韩文等 |
| 切换生效 | 立即生效，无需重启 |
| 持久化 | 重启后保持设置 |

### 2.2 主题设置

| 规则项 | 规则描述 |
|-------|---------|
| 模式 | 浅色、深色、跟随系统 |
| 切换生效 | 立即生效 |
| 持久化 | 重启后保持设置 |

### 2.3 安全设置

| 设置项 | 规则 |
|-------|------|
| 密码 | 支持修改钱包密码 |
| 生物识别 | 支持指纹/Face ID |
| 自动锁定 | 支持设置自动锁定时间 |
| 截屏保护 | 敏感页面禁止截屏 |

### 2.4 网络设置

| 规则项 | 规则描述 |
|-------|---------|
| 自定义 RPC | 支持添加自定义 RPC 节点 |
| 节点切换 | 支持切换 RPC 节点 |
| 网络检测 | 显示当前网络状态 |

## 3. 通知模块规则

### 3.1 推送通知

| 规则项 | 规则描述 |
|-------|---------|
| 订阅 | 支持开启/关闭推送 |
| 分类 | 按类型订阅（交易、价格、系统等） |
| 权限 | 需要系统推送权限 |

### 3.2 消息中心

| 规则项 | 规则描述 |
|-------|---------|
| 列表 | 显示所有通知消息 |
| 已读/未读 | 标记已读状态 |
| 清空 | 支持清空全部/单条 |
| 跳转 | 点击跳转到对应详情 |

### 3.3 通知状态机

```
生成 → 推送 → 到达 → 点击跳转 → 已读
              ↓
          离线堆积 → 去重
```

### 3.4 Deep Link 规则

| 规则项 | 规则描述 |
|-------|---------|
| 跳转解析 | 解析通知中的跳转参数 |
| 失效兜底 | 链接失效时跳转到列表页 |
| 参数校验 | 校验跳转参数有效性 |

## 6. 扫码规则

### 6.1 支持的二维码类型

| 类型 | 处理方式 |
|-----|---------|
| 钱包地址 | 跳转转账页，填入地址 |
| WalletConnect | 发起连接 |
| 交易请求 | 显示交易确认页 |
| 普通链接 | 打开内置浏览器 |

### 6.2 权限处理

| 场景 | 规则 |
|-----|------|
| 首次使用 | 请求相机权限 |
| 权限拒绝 | 引导到设置页开启 |
| 权限已开启 | 直接打开扫码 |

### 6.3 扫码识别

| 规则项 | 规则描述 |
|-------|---------|
| 实时识别 | 对准即识别 |
| 识别失败 | 提示重新扫描 |
| 不支持类型 | 提示不支持的二维码 |

## 8. App 升级规则

### 8.1 版本检测

| 规则项 | 规则描述 |
|-------|---------|
| 启动检测 | App 启动时检测新版本 |
| 手动检测 | 设置中手动检测 |
| 更新提示 | 新版本弹窗提示 |

### 8.2 升级类型

| 类型 | 规则 |
|-----|------|
| 强制升级 | 必须升级才能使用 |
| 推荐升级 | 可跳过，后续提醒 |
| 静默升级 | 后台下载，提示安装 |

### 8.3 升级流程

| 步骤 | 规则 |
|-----|------|
| 下载 | 显示下载进度 |
| 安装 | 跳转应用商店/直接安装 |
| 失败处理 | 提示重试 |

## 9. 规则维护指南

### 如何更新规则

1. **发现规则变更**：
   - 在测试过程中发现规则与文档不一致
   - 收到产品/开发通知规则变更

2. **更新文档**：
   - 直接修改对应章节
   - 在变更记录中记录更新时间和原因
