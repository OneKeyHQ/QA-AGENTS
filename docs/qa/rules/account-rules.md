# Account 模块测试规则文档

> 本文档记录账户模型模块的核心测试规则。
> 生成账户模型相关测试用例时，必须参考本文档。

---

## 1. 账户类型规则

### 1.1 支持的账户类型

| 账户类型 | 英文名称 | 说明 | 能力限制 |
|---------|---------|------|---------|
| HD 钱包 | HD Wallet | 通过助记词派生的钱包 | 完整功能 |
| 硬件钱包 | Hardware Wallet | 通过硬件设备管理的钱包 | 需设备确认 |
| 观察账户 | Watched Only Account | 仅查看余额，无私钥 | 不支持签名/转账 |
| 外部账户 | External Account | 第三方钱包导入 | 受账户系列限制 |
| 私钥账户 | Private Key Account | 通过私钥导入 | 受账户系列限制 |

### 1.2 账户能力矩阵

| 功能 | HD 钱包 | 硬件钱包 | 观察账户 | 外部账户 | 私钥账户 |
|-----|--------|---------|---------|---------|---------|
| 查看余额 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 转账 | ✓ | ✓ | ✗ | ✓ | ✓ |
| Swap | ✓ | ✓ | ✗ | ✓ | ✓ |
| DApp 签名 | ✓ | ✓ | ✗ | ✓ | ✓ |
| 导出私钥 | ✓ | ✗ | ✗ | ✗ | ✓ |
| 导出助记词 | ✓ | ✗ | ✗ | ✗ | ✗ |

## 2. 账户派生规则

### 2.1 BIP 标准支持

| BIP 标准 | 路径格式 | 适用链 | 地址类型 |
|---------|---------|-------|---------|
| BIP44 | m/44'/coin'/0'/0/index | 所有链 | Legacy |
| BIP49 | m/49'/coin'/0'/0/index | BTC | P2SH-SegWit |
| BIP84 | m/84'/coin'/0'/0/index | BTC | Native SegWit |
| BIP86 | m/86'/coin'/0'/0/index | BTC | Taproot |

### 2.2 派生一致性规则

- **同助记词派生**：同一助记词在不同设备/平台派生的地址必须一致
- **多账户派生**：支持派生多个账户（index 递增）
- **链间隔离**：不同链的派生路径相互独立

## 3. 助记词管理规则

### 3.1 创建流程

| 步骤 | 规则 | 验证点 |
|-----|------|-------|
| 生成 | 支持 12 词 | 导出词数正确 |
| 展示 | 逐词展示或整体展示 | 截屏限制生效 |
| 备份确认 | 用户确认已备份 | 勾选确认框 |
| 验证 | 随机抽取词验证 | 验证顺序正确 |

### 3.2 导入流程

| 场景 | 规则 | 错误提示 |
|-----|------|---------|
| 格式错误 | 非 BIP39 词库 | "无效助记词" |
| 词数错误 | 非 12/15/18/21/24 词 | "无效助记词" |
| 校验和错误 | 最后一词校验失败 | "无效助记词" |
| 重复导入 | 已存在相同钱包 | "该钱包已存在" |

### 3.3 安全规则

- **截屏限制**：助记词页面禁止截屏
- **剪贴板清除**：在助记词/私钥输入框粘贴后自动清除剪贴板
- **掩码显示**：默认掩码，点击显示
- **二次验证**：导出需密码/生物识别确认

## 4. 私钥管理规则

### 4.1 私钥格式支持

| 链类型 | 私钥格式 | 长度 |
|-------|---------|-----|
| EVM | 0x 开头十六进制 | 64 字符（不含 0x） |
| BTC | WIF 格式 | 51-52 字符 |
| Solana | Base58 | 87-88 字符 |

### 4.2 导入规则

- **格式校验**：自动识别私钥格式
- **链匹配**：私钥格式必须与目标链匹配
- **重复检测**：导入已存在的私钥定位到对应账户并提示"该钱包已存在"

### 4.3 导出规则

- **权限验证**：需密码/生物识别确认
- **安全提示**：显示风险警告
- **一次性显示**：退出后需重新验证才能再次显示

### 4.4 私钥导出/导入细则

| 规则项 | 规则描述 |
|-------|---------|
| 网络/派生路径 | 可选，默认与钱包首页选择一致 |
| 未创建地址 | 选择尚未创建地址的网络/路径时，私钥栏为空，提示该账户还没有创建地址 |
| 已创建地址 | 私钥栏掩码显示，可切换可见，可复制 |
| 复制确认 | 用户勾选确认信息前无法确认复制 |
| 导入预测 | 根据输入字符预测匹配网络 |
| 找不到网络 | 点击「找不到想要的网络」可展开所有网络列表选择 |
| BTC/fork 链 | 导入私钥的 BTC 及 fork 链账户可以导出对应公钥 |

### 4.5 公钥导出规则

| 规则项 | 规则描述 |
|-------|---------|
| 入口 | 账户选择器 → 第一个账户三个点 → 查看详情 → 导出公钥 |
| 权限 | 私钥/HD钱包导出需验证密码，公钥观察账户导出不需要验证密码 |
| 适用 | BTC 及 fork 链的HD/HW/私钥/公钥观察账户 |

## 5. 外部账户规则

### 5.1 WalletConnect 连接

| 规则项 | 规则描述 |
|-------|---------|
| 流程 | onboarding → 点击 WalletConnect → 弹出WalletConnect弹窗 → 其他设备外部钱包扫码/搜索本机外部钱包 → 在外部钱包客户端确认连接 → 创建外部钱包账户 |
| 账户命名 | 外部钱包名称 + 序号 |
| 显示校验 | 账户连接钱包图标、网络图标、地址正确 |
| 单连接 | 每次点击 WalletConnect 会断开之前已连接的 WalletConnect 钱包，只显示最新连接的账户 |

### 5.2 注入连接（Web/Ext）

| 规则项 | 规则描述 |
|-------|---------|
| 常驻显示 | OneKey/Phantom/Coinbase/OKX 常驻显示 |
| 其他插件 | 安装后显示，未安装不显示 |
| 未安装 | 未安装的插件钱包跳转对应 Chrome 商店页 |
| 已安装 | 已安装的插件钱包注入连接对应账户 |

### 5.3 连接异常/变更

| 场景 | 规则 |
|-----|------|
| 外部钱包拒绝 | 提示拒绝连接，重试可再次发起并成功创建账户 |
| 不删除注入账户，修改外部钱包网络和地址后重新导入 | 网络图标和地址更新，名称和序号保持不变 |
| 删除已注入账户，修改外部钱包后重新导入 | 网络图标和地址更新，名称保持不变，序号增加 |
| 禁用第三方插件 | 禁用后的其他外部钱包不出现在注入钱包选择列表中，常驻显示的OneKey/Phantom/Coinbase/OKX 跳转对应 Chrome 商店页 |
| 相同地址 | 可重复连接 |


### 5.4 Web 端特殊规则

| 规则项 | 规则描述 |
|-------|---------|
| 钱包页 | 不显示钱包页及网络选择器 |
| 网络锁定 | 锁定网络为 All networks |
| 市场搜索 | 搜索中隐藏「账户」「我的资产」「dApps」分类及搜索结果 |
| 设置 | 隐藏所有与本地账户相关的设置项 |
| URL 钱包返回 | 点击返回按钮返回市场页 |

## 6. 硬件钱包规则

### 6.1 设备与连接方式

| 设备 | 支持连接方式 | 端限制 |
|-----|-------------|-------|
| Pro | USB / 蓝牙 / 二维码 | 全端 |
| Touch | USB / 蓝牙 | 全端 |
| Classic / 1S | USB / 蓝牙 | 全端 |
| Mini | USB | 移动端不显示 Mini，不显示蓝牙分类 |

| 端 | 展示分类 |
|---|---------|
| 桌面端（除Linux） | USB / 蓝牙 / 二维码 |
| 移动端 | 蓝牙 / 二维码 |
| 插件端/Linux桌面端 | USB / 二维码 |

### 6.2 连接流程

| 规则项 | 规则描述 |
|-------|---------|
| USB | 直接展示连接中的设备列表 |
| 蓝牙 | 直接开始搜索并展示可连接的设备 |
| 停留 10 秒 | 展示「设备连接有问题吗？」按钮，点击打开建议解决方案 |
| 拔掉 USB | 设备信息清除 |
| 关闭蓝牙 | 提示蓝牙关闭和跳转按钮 |
| 固件升级 | 可跳过；桌面端蓝牙连接升级时提示插入 USB 线 |

### 6.3 验真与 PIN

| 场景 | 规则 |
|-----|------|
| 验真取消 | 硬件上点击「取消」→ 回到 Onboarding 页面，提示操作取消 |
| 验真失败 | 设备验真不通过，App 弹窗提示设备异常，提供 Contact us 跳转 |
| 验真中断 | 断网/断开设备连接（拔出usb线，关闭蓝牙，关闭设备） → 提示硬件通信失败，恢复后可重试 |
| 跳过验真 | 验真中断后选择仍要继续 → 跳过验真，创建钱包后在设备信息中标识未验真 |
| PIN 错误（Classic/Mini/1S） | App 端提示 PIN 码错误 |
| PIN 错误（Pro/Touch） | 10 次错误后设备被重置，再次连接进入激活页面 |

### 6.4 Passphrase 与隐藏钱包

| 设备状态 | 创建选项 |
|---------|---------|
| 未开启 passphrase，未创建过标准钱包 | 直接创建标准钱包 |
| 开启 passphrase，未创建过标准钱包 | 显示创建标准钱包/隐藏钱包选项 |
| 开启 passphrase，已创建过标准钱包 | 直接提示输入 Passphrase 创建隐藏钱包 |
| 开启 passphrase，重置前创建过标准钱包 | 显示创建标准钱包/隐藏钱包选项 |

| 场景 | 规则 |
|-----|------|
| 未开启 passphrase | 无法创建隐藏钱包，需先启用；启用后钱包选择器显示快捷添加选项 |
| 未开启时操作 | 核对地址/创建地址/转账/swap 均提示打开 passphrase |
| 快捷添加按钮 | 标准钱包+已开启 passphrase 或 有隐藏钱包时显示；二维码钱包视为未开启 passphrase |

### 6.5 重复创建规则

| 场景 | 规则 |
|-----|------|
| 相同设备+相同助记词 | 创建并更新覆盖旧钱包，已验证则不需再次验证 |
| 不同设备+相同助记词 | 创建新钱包 |
| 相同设备+重置使用不同助记词 | 创建新钱包，旧钱包标记废弃 |
| 相同设备+重置使用相同助记词，新连接首先创建隐藏钱包 或 旧钱包没有标准钱包 | 创建新钱包，旧钱包标记废弃 |
| 相同设备+重置使用相同助记词，新连接首先创建标准钱包 且 旧钱包有标准钱包 | 绑定旧钱包，去掉废弃标记 |


### 6.6 Passphrase 保持连接

| 规则项 | 规则描述 |
|-------|---------|
| 默认 | 创建隐藏钱包时「保持连接」开关默认开启 |
| 关闭后重启 | 关闭「保持连接」的隐藏钱包在重启 APP 后被隐藏，从新创建后保留之前的账户创建和修改 |
| 相同 passphrase | 提示创建相同账户，定位到隐藏钱包并根据创建时选择更新开关状态 |

### 6.7 Passphrase 输入规则

| 场景 | 规则 |
|-----|------|
| 不输入 | APP禁止不输入，可切换到设备上操作，创建成功与标准钱包地址一致 |
| 中文/emoji | 提示不支持字符 |
| 英文特殊字符 @#$%^&*()_+ | 创建成功 |
| 空格 | 创建成功 |
| 超长（>50 字符） | 提示输入过长 |
| 过期 session | 输入错误 passphrase 提示不匹配，正确可完成操作 |

## 7. 观察账户规则

### 7.1 导入方式

| 方式 | 规则 |
|-----|------|
| 地址 | 支持不同网络地址，选择对应网络导入 |
| 域名 | 支持不同后缀/网络域名，选择对应网络并解析后导入 |
| 公钥 | 支持不同网络公钥，选择对应网络导入 |

### 7.2 异常与校验

| 场景 | 规则 |
|-----|------|
| 输入为空 | 按钮置灰不可点击 |
| 非法字符/无效地址/无效域名 | 提示错误 |
| 网络与地址不匹配 | 提示错误 |
| 未激活 Cardano 地址 | 地址校验正常，导入时提示账户未激活 |
| BTC xpub 不同派生路径 | 正常导入不同地址，导出公钥与导入一致 |
| 重复导入地址 | 直接定位到已导入的观察账户 |
| 导入相同地址对应公钥 | 可导入，公钥导入的观察账户可导出公钥 |
| 地址前后空格 | 过滤空格，正常导入 |
| 地址中间空格 | 提示错误 |

## 8. Web URL 钱包规则

### 8.1 搜索访问

| 规则项 | 规则描述 |
|-------|---------|
| 搜索框 | 输入地址自动发起检索，检索中显示灰色占位符 |
| 完全匹配 | 展示网络图标/网络名称/钱包地址 |
| 点击结果 | URL 转为 1key.so/url-account/{网络}/{地址}；web 端改变网址，桌面/ext 端进入二级页面 |
| EVM 网络 | 网络选择器可切换其他 EVM 网络 |
| 非 EVM 网络 | 网络选择器不可用 |
| Tab | 投资组合/NFT/历史/授权 tab 切换显示 |
| 理财/交易 | 切换回钱包账户，展示对应资产/交易信息 |

### 8.2 链接地址访问

| URL 类型 | 预期 |
|---------|------|
| 正确 URL（网络与地址匹配） | 打开对应 web URL 钱包 |
| 地址与网络不匹配 | 显示错误提示页面 |
| 地址错误 | 显示错误提示页面 |
| 网络错误 | 显示错误提示页面 |
| evm/0x... | 打开 EVM 地址对应的 All Network 钱包页 |

### 8.3 浏览器兼容

支持 Chrome、Safari、Edge、Firefox、OneKey 内置浏览器。

## 9. 钱包账户管理规则

### 9.1 创建钱包

| 规则项 | 规则描述 |
|-------|---------|
| 名称 | 支持自定义名称，默认"钱包 N" |
| 头像 | 支持选择/自定义头像 |
| 密码 | 创建时设置钱包密码 |
| 备份提醒 | 未备份时显示提醒标识 |

### 9.2 切换钱包/账户

| 场景 | 规则 |
|-----|------|
| 切换钱包 | 资产页面刷新，显示新钱包资产 |
| 切换账户 | 同钱包内切换派生账户 |
| 切换链 | 保持当前账户，切换显示链 |

### 9.3 删除/重置

| 操作 | 规则 | 安全措施 |
|-----|------|---------|
| 删除单链钱包账户 | 仅删除单个账户 | 私钥账户二次确认，观察/外部账户直接删除 |
| 删除多链钱包账户 | 仅删除派生账户 | 每次启动APP后第一次移除进行提示 |
| 删除钱包 | 删除整个钱包（含所有账户） | 警告提示 + 校验内存密码 |
| 重置 App | 清除所有数据 | 多重确认 |

## 10. 通用规则

### 状态机
- 创建/导入 → 备份校验 → 解锁/锁定 → 切链/切账户 → 恢复

### 资金一致性
- 资产页/详情页/交易记录三处一致
- 隐藏资产功能不影响实际余额

### 权限/安全
- 助记词/私钥页截屏限制
- 剪贴板清除
- 导出权限
- biometric 失败分支处理

### 链上不确定性
- 多 RPC 切换
- 链切换中断恢复
- 地址派生一致性

### 风控
- 高风险钱包提示
- 观察钱包 vs 冷钱包能力差异限制

### 可观测
- 错误码、导入失败原因可解释（格式/校验和/路径）

## 11. 规则维护指南

### 如何更新规则

1. **发现规则变更**：
   - 在测试过程中发现规则与文档不一致
   - 收到产品/开发通知规则变更
   - 新增账户类型或派生标准

2. **更新文档**：
   - 直接修改对应章节
   - 在变更记录中记录更新时间和原因
