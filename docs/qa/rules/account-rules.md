# Account 模块测试规则文档

> 本文档记录账户模型模块的核心测试规则。
> 生成账户模型相关测试用例时，必须参考本文档。

---

## 1. 账户类型规则

### 1.1 支持的账户类型

| 账户类型 | 英文名称 | 说明 | 能力限制 |
|---------|---------|------|---------|
| HD 钱包 | HD Wallet | 通过助记词派生的钱包 | 完整功能 |
| 硬件钱包 | Hardware Wallet | 通过硬件设备管理的钱包 | 需设备确认 |
| 观察账户 | Watched Only Account | 仅查看余额，无私钥 | 不支持签名/转账 |
| 外部账户 | External Account | 第三方钱包导入 | 受账户系列限制 |
| 私钥账户 | Private Key Account | 通过私钥导入 | 受账户系列限制 |

### 1.2 账户能力矩阵

| 功能 | HD 钱包 | 硬件钱包 | 观察账户 | 外部账户 | 私钥账户 |
|-----|--------|---------|---------|---------|---------|
| 查看余额 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 转账 | ✓ | ✓ | ✗ | ✓ | ✓ |
| Swap | ✓ | ✓ | ✗ | ✓ | ✓ |
| DApp 签名 | ✓ | ✓ | ✗ | ✓ | ✓ |
| 导出私钥 | ✓ | ✗ | ✗ | ✗ | ✓ |
| 导出助记词 | ✓ | ✗ | ✗ | ✗ | ✗ |

## 2. 账户派生规则

### 2.1 BIP 标准支持

| BIP 标准 | 路径格式 | 适用链 | 地址类型 |
|---------|---------|-------|---------|
| BIP44 | m/44'/coin'/0'/0/index | 所有链 | Legacy |
| BIP49 | m/49'/coin'/0'/0/index | BTC | P2SH-SegWit |
| BIP84 | m/84'/coin'/0'/0/index | BTC | Native SegWit |
| BIP86 | m/86'/coin'/0'/0/index | BTC | Taproot |

### 2.2 派生一致性规则

- **同助记词派生**：同一助记词在不同设备/平台派生的地址必须一致
- **多账户派生**：支持派生多个账户（index 递增）
- **链间隔离**：不同链的派生路径相互独立

## 3. 助记词管理规则

### 3.1 创建流程

| 步骤 | 规则 | 验证点 |
|-----|------|-------|
| 生成 | 支持 12/18/24 词 | 词数选择正确 |
| 展示 | 逐词展示或整体展示 | 截屏限制生效 |
| 备份确认 | 用户确认已备份 | 勾选确认框 |
| 验证 | 随机抽取词验证 | 验证顺序正确 |

### 3.2 导入流程

| 场景 | 规则 | 错误提示 |
|-----|------|---------|
| 格式错误 | 非 BIP39 词库 | "助记词格式错误" |
| 词数错误 | 非 12/18/24 词 | "助记词数量不正确" |
| 校验和错误 | 最后一词校验失败 | "助记词校验失败" |
| 重复导入 | 已存在相同钱包 | "该钱包已存在" |

### 3.3 安全规则

- **截屏限制**：助记词页面禁止截屏
- **剪贴板清除**：复制后自动清除剪贴板（60 秒内）
- **掩码显示**：默认掩码，点击显示
- **二次验证**：导出需密码/生物识别确认

## 4. 私钥管理规则

### 4.1 私钥格式支持

| 链类型 | 私钥格式 | 长度 |
|-------|---------|-----|
| EVM | 0x 开头十六进制 | 64 字符（不含 0x） |
| BTC | WIF 格式 | 51-52 字符 |
| Solana | Base58 | 87-88 字符 |

### 4.2 导入规则

- **格式校验**：自动识别私钥格式
- **链匹配**：私钥格式必须与目标链匹配
- **重复检测**：禁止导入已存在的私钥

### 4.3 导出规则

- **权限验证**：需密码/生物识别确认
- **安全提示**：显示风险警告
- **一次性显示**：退出后需重新验证才能再次显示

## 5. 钱包账户管理规则

### 5.1 创建钱包

| 规则项 | 规则描述 |
|-------|---------|
| 名称 | 支持自定义名称，默认"钱包 N" |
| 头像 | 支持选择/自定义头像 |
| 密码 | 创建时设置钱包密码 |
| 备份提醒 | 未备份时显示提醒标识 |

### 5.2 切换钱包/账户

| 场景 | 规则 |
|-----|------|
| 切换钱包 | 资产页面刷新，显示新钱包资产 |
| 切换账户 | 同钱包内切换派生账户 |
| 切换链 | 保持当前账户，切换显示链 |

### 5.3 删除/重置

| 操作 | 规则 | 安全措施 |
|-----|------|---------|
| 删除账户 | 仅删除派生账户 | 二次确认 |
| 删除钱包 | 删除整个钱包（含所有账户） | 密码确认 + 警告提示 |
| 重置 App | 清除所有数据 | 多重确认 |

## 6. 通用规则

### 状态机
- 创建/导入 → 备份校验 → 解锁/锁定 → 切链/切账户 → 恢复

### 资金一致性
- 资产页/详情页/交易记录三处一致
- 隐藏资产功能不影响实际余额

### 权限/安全
- 助记词/私钥页截屏限制
- 剪贴板清除
- 导出权限
- biometric 失败分支处理

### 链上不确定性
- 多 RPC 切换
- 链切换中断恢复
- 地址派生一致性

### 风控
- 高风险钱包提示
- 观察钱包 vs 冷钱包能力差异限制

### 可观测
- 错误码、导入失败原因可解释（格式/校验和/路径）

   - 收到产品/开发通知规则变更
   - 新增账户类型或派生标准

2. **更新文档**：
   - 直接修改对应章节
   - 在变更记录中记录更新时间和原因
